# 质量控制检查清单

本文档定义喵娘分析技能的输出前专项质量检查标准。

---

## 检查流程概述

每个热点主题报告在输出前，必须通过**全部专项质量检查**。

---

## 1. 态势评级专项检查

### 检查项
- [ ] **标准性**：是否使用了五种标准评级之一？
  - ⚡⚡⚡⚡⚡核爆级
  - ⚡⚡⚡⚡沸腾级
  - ⚡⚡⚡发酵级
  - ⚡⚡观望级
  - ⚠️高风险预警级
- [ ] **准确性**：评级是否符合知识库判定标准？
- [ ] **完整性**：态势评级是否包含三部分？
  - 知识库标准态势评级
  - 喵娘锐评（50字内精炼点评）
  - 风险等级（🟢/🟡/🟠/🔴）

---

## 2. 信源标注专项检查

### 检查项
- [ ] **标识标准**：是否使用标准标识？
  - [🟢S级] 一级信源（官方机构）
  - [🔵A级] 二级信源（权威媒体）
  - [🟡B级] 三级信源（专业财经媒体）
  - [🟠C级] 四级信源（数据服务商）
- [ ] **格式完整**：是否包含完整格式？
  - `[信源等级标识] [YYYY-MM-DD]-信源名称：[信息精要] （对市场生态的冲击解读） {时效标识}`
- [ ] **冲击解读**：每个信源是否包含"对市场生态的冲击解读"？
- [ ] **时效标识**：每个信源是否有时效标识？
  - 🆕 实时（0-6小时）
  - 📅 当日（6-24小时）
  - ⏰ 隔日（24-72小时）
  - ❌ 过期（>72小时）
- [ ] **排序合规**：信源是否按等级+时间规则排序？
  - 等级排序：S级→A级→B级→C级
  - 时间排序：同等级内时间倒序（最新优先）

---

## 3. 作战地图专项检查

### 检查项
- [ ] **元素完整**：是否包含四个完整元素？
  - 核心突击位
  - 侧翼观测位
  - 关键路标
  - 风险路标
- [ ] **核心突击位**：是否标注"风险可控"？
- [ ] **侧翼观测位**：是否标注"需进一步验证"？
- [ ] **路标有效性**：关键路标和风险路标是否可验证？

---

## 4. 风险预警专项检查

### 检查项
- [ ] **等级准确**：风险等级是否按风险信号数量准确评估？

| 风险信号数量 | 风险等级 |
|--------------|----------|
| 0个 | 🟢低风险 |
| 1-2个 | 🟡中风险 |
| 3-4个 | 🟠高风险 |
| 5个及以上 | 🔴极高风险 |

---

## 5. 喵娘洞见专项检查

### 检查项
- [ ] **维度完整**：是否包含全部六个维度？
  - 生态位变化
  - 主力剧本推演
  - 节点必然性
  - 风险边界分析
  - 国内外联合共振分析
  - 多重驱动分析
- [ ] **实质性**：每个维度是否提供实质性分析？
- [ ] **避免模板化**：是否避免了空洞的模板化表述？

---

## 6. 全局统计检查

### 检查项
- [ ] **全面覆盖**：是否输出所有评分≥70分的热点？
- [ ] **无遗漏**：热点发现状态是否准确反映实际发现数量？
- [ ] **共振识别**：国内外联合共振热点是否正确识别？
- [ ] **驱动识别**：多重驱动热点是否正确识别？

---

## 步骤执行与质量控制集成检查清单

### 步骤1：周末消息面扫描
- [ ] 第0层市场全景扫描完成
- [ ] 第1层官方确认层完成
- [ ] 潜在热点清单输出

### 步骤2：建立逻辑链条
- [ ] 逻辑链条清晰完整
- [ ] 因果关系明确
- [ ] 符合市场运行逻辑

### 步骤3：验证市场预期
- [ ] 资金情绪验证完成
- [ ] 板块热度验证完成
- [ ] 市场关注度验证完成
- [ ] 国内外联合共振验证完成

### 步骤4：评估舆情热度
- [ ] 媒体报道数量评估
- [ ] 社交媒体讨论热度评估
- [ ] 投资者讨论活跃度评估

### 步骤5：综合评分
- [ ] 消息面驱动得分≥35分
- [ ] 市场预期得分≥20分
- [ ] 舆情热度得分≥10分
- [ ] 总分≥70分

---

## 终极检查

**每个热点必须通过以上全部检查项**才能输出。

如有任何检查项未通过，必须返回后台重新分析，直到通过为止。
