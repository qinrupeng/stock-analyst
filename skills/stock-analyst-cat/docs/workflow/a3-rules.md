# A3混合模式规则

> 定义何时用AI快速出、何时用模板深度分析
> 版本：1.0 (2026-03-14)

---

## 核心规则

### 规则1：默认快速模式

**触发条件**：
- 用户没有特别说明"深度分析"
- 热点数量≤2个
- 时间紧张

**执行流程**：
```
1. 搜索热点
2. AI快速评分
3. 直接输出结果
4. 用简化审查（7项）
```

**预计时间**：5分钟

---

### 规则2：深度分析模式

**触发条件**：
- 用户明确说"深度分析"
- 热点数量≥3个
- 用户要求"全面分析"

**执行流程**：
```
1. 搜索热点
2. 完整评分模板
3. 详细分析
4. 完整审查清单
5. 格式化输出
```

**预计时间**：15-20分钟

---

### 规则3：始终审查

**无论哪种模式**，都必须：
- 用简化审查清单（7项）快速检查
- 记录问题
- 给出结论

---

## 决策树

```
用户输入
    │
    ▼
是否说"深度分析"？
    │
    ├─ 是 → 深度分析模式 → 完整模板
    │
    └─ 否 → 热点数量≥3？
              │
              ├─ 是 → 深度分析模式 → 完整模板
              │
              └─ 否 → 快速模式 → 简化输出
                        │
                        ▼
                    简化审查（7项）
```

---

## 简化版 vs 完整版对比

| 维度 | 快速模式 | 深度模式 |
|------|----------|----------|
| 评分 | 口头估算 | 完整模板 |
| 分析 | 重点突出 | 详细全面 |
| 审查 | 7项简化 | 完整清单 |
| 输出 | 精简版 | 完整版 |
| 时间 | 5分钟 | 20分钟 |

4. 详细分析

## 核心原则

1. **默认快速**：除非用户要求，否则默认快速
2. **质量底线**：无论哪种模式，审查不能跳过
3. **用户优先**：用户时间优先，但质量有保障

---

## 审查清单（简化7项）

| # | 检查项 |
|---|--------|
| 1 | 态势评级≥75分 |
| 2 | S1+S2+A级信源≥70% |
| 3 | 核心突击位有股票 |
| 4 | 侧翼观测位有股票 |
| 5 | 有风险预警 |
| 6 | 有具体阈值 |
| 7 | 有仓位建议 |
