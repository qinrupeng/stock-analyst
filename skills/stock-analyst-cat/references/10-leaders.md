# 龙头识别与热点管理

本文档定义板块龙头识别机制和热点白名单管理。

---

## 龙头识别机制

### 龙头定义
板块中具有**最强基本面支撑**、**最大资金关注度**、**最明确上涨逻辑**的领涨标的。

### 龙头识别标准

#### 基本面维度
| 标准 | 分值 | 说明 |
|------|------|------|
| 业绩支撑 | 20分 | 业绩增长确定性高 |
| 行业地位 | 15分 | 行业龙头或细分领域第一 |
| 资源禀赋 | 10分 | 拥有核心资源或技术 |
| 成长性 | 10分 | 未来增长空间大 |

#### 市场维度
| 标准 | 分值 | 说明 |
|------|------|------|
| 资金关注度 | 20分 | 主力资金持续流入 |
| 涨幅领先 | 15分 | 领涨板块，涨幅靠前 |
| 成交量 | 10分 | 成交量放大，资金活跃 |
| 涨停次数 | 10分 | 涨停次数多，龙头属性强 |

#### 逻辑维度
| 标准 | 分值 | 说明 |
|------|------|------|
| 逻辑清晰 | 20分 | 上涨逻辑最直接、最明确 |
| 不可替代 | 15分 | 逻辑难以被替代或复制 |
| 预期差大 | 10分 | 市场预期差大，超预期空间大 |
| 催化剂多 | 10分 | 后续催化剂多，持续性强 |

### 龙头评分公式
```
龙头综合得分 = 基本面得分 + 市场得分 + 逻辑得分
```

### 龙头分类

| 龙头类型 | 特征 | 参与策略 |
|----------|------|----------|
| 总龙头 | 板块中涨幅最大、逻辑最硬 | 重仓参与，持有为主 |
| 补涨龙头 | 逻辑类似总龙头，涨幅落后 | 适度参与，博取补涨 |
| 情绪龙头 | 情绪溢价高，基本面一般 | 快进快出，不宜恋战 |
| 趋势龙头 | 趋势性上涨，走势稳健 | 回调低吸，趋势持有 |

## 热点白名单管理机制

### 入池标准

#### 必须满足
1. **总分≥70分**：达到推荐门槛
2. **驱动类型明确**：多重驱动优先
3. **龙头明确**：有明确的板块龙头
4. **逻辑清晰**：逻辑链条完整

#### 加分项
1. **国内外联合共振**：+10分
2. **情绪到位+催化剂到位**：+10分
3. **资金持续流入**：+5分
4. **机构买入**：+5分

### 出池标准

#### 触发条件
1. **热度下降**：涨停股减少，资金流出
2. **逻辑证伪**：关键逻辑被证伪
3. **风险暴露**：出现重大风险信号
4. **评级下调**：态势评级下调至观望级以下

### 动态调整

#### 每日更新
1. 根据最新信息更新白名单
2. 调整热点评级
3. 更新推荐优先级

#### 实时监控
1. 盘中持续监控热点变化
2. 捕捉突发因素
3. 及时预警风险

#### 及时调整
1. 根据市场变化及时调整推荐
2. 修正作战地图
3. 更新风险提示

## 热点生命周期管理

### 萌芽期
- **特征**：媒体报道<5篇/日，涨停股<3只，资金净流入<10亿元
- **策略**：关注，跟踪验证
- **操作**：小仓位试探

### 爆发期
- **特征**：媒体报道>20篇/日，涨停股>10只，资金净流入>50亿元
- **策略**：积极参与，抓住主升
- **操作**：重仓参与，趋势持有

### 成熟期
- **特征**：涨停股数量稳定，资金流入放缓，板块涨幅>15%
- **策略**：谨慎参与，关注补涨
- **操作**：减仓止盈，关注补涨

### 衰退期
- **特征**：资金净流出，跌停股>3只，负面消息增多
- **策略**：观望为主，回避风险
- **操作**：清仓离场，回避风险
