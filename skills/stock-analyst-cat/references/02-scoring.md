# 评分机制与态势评级

本文档定义喵娘分析技能的量化评分标准和态势评级体系。

---

## 评分机制

### 基础评分要素
| 要素 | 分值范围 | 评估维度 |
|------|----------|----------|
| 政策支持力度 | 0-30分 | 政策级别、落地确定性、资金支持力度 |
| 技术突破程度 | 0-25分 | 技术领先性、商业化前景、产业链影响 |
| 市场情绪热度 | 0-20分 | 媒体报道量、社交媒体讨论、板块涨跌 |
| 资金流向 | 0-15分 | 主力资金净流入、北向资金流向 |
| 时间窗口 | 0-10分 | 催化剂时效性、事件节点临近程度 |

### 多重驱动评分公式
```
多重驱动评分 = 政策驱动(20%) + 技术驱动(20%) + 市场驱动(20%) + 国外驱动(20%) + 情绪驱动(20%)
```
- ≥90分：史诗级多重驱动
- ≥70分：多重驱动
- ≥50分：单驱动

---

## 情绪到位判断标准

满足以下**3个及以上**条件即为"情绪到位"：

| 条件 | 阈值 |
|------|------|
| 市场关注度 | 媒体报道数量日增≥10篇 |
| 资金情绪 | 板块资金连续3日流入 |
| 板块热度 | 涨停股票数量≥5只 |
| 市场预期 | 机构研报数量日增≥5篇 |

---

## 情绪+催化剂爆发式上涨识别机制

### 识别条件
**情绪到位 + 催化剂到位 = 爆发式上涨**

### 情绪到位量化标准
1. **市场关注度**（10分）：媒体报道数量日增≥10篇
2. **资金情绪**（10分）：板块资金连续3日流入
3. **板块热度**（10分）：涨停股票数量≥5只
4. **市场预期**（10分）：机构研报数量日增≥5篇
5. **社交媒体热度**（5分）：社交媒体讨论热度日增≥1000条
6. **投资者讨论**（5分）：论坛、股吧讨论热度提升

**情绪得分≥30分 = 情绪到位**

### 催化剂到位量化标准
1. **国外技术突破**（25分）：马斯克/OpenAI等发布重大技术进展
2. **重大政策发布**（25分）：证监会/发改委/工信部等发布重大政策
3. **重大技术突破**（25分）：新技术研发成功、商业化应用突破
4. **重大事件**（25分）：重大业绩、超预期订单等

**催化剂得分≥25分 = 催化剂到位**

---

## 国内外联合共振识别机制

### 共振定义
**国外技术突破 + 国内跟进 = 国内外联合共振**

### 共振条件
1. **国外驱动**（40分）：有明确的国外技术突破
2. **国内政策跟进**（30分）：有国内政策或市场跟进反应
3. **产业链联动**（30分）：国内外产业链形成联动

**共振得分≥70分 = 国内外联合共振**

### 共振类型
1. **技术共振型**：国外技术突破 → 国内技术跟进 → 产业链爆发
2. **政策共振型**：国外政策变化 → 国内政策调整 → 市场反应
3. **市场共振型**：国外市场变化 → 国内市场联动 → 情绪传导

## 多重驱动分析机制

### 多重驱动定义
**政策、技术、市场、国外、情绪中至少2个驱动类型 = 多重驱动**

### 多重驱动评分矩阵
| 驱动类型 | 分值 | 评估要点 |
|----------|------|----------|
| 政策驱动 | 20分 | 政策级别、落地确定性、资金支持力度 |
| 技术驱动 | 20分 | 技术领先性、商业化前景、产业链影响 |
| 市场驱动 | 20分 | 供需变化、价格波动、成交量变化 |
| 国外驱动 | 20分 | 国际市场变化、技术突破传导 |
| 情绪驱动 | 20分 | 市场情绪热度、投资者预期 |

### 多重驱动强度判定
- ≥90分：史诗级多重驱动（所有驱动同时发力）
- ≥70分：强多重驱动（4个以上驱动发力）
- ≥50分：中等多重驱动（3个驱动发力）
- ≥30分：弱多重驱动（2个驱动发力）

## 四阶段动态评分机制

### 阶段定义与评分权重

| 阶段 | 时间窗口 | 评分特点 |
|------|----------|----------|
| 盘前阶段 | 9:00-9:30 | 侧重预期评估、信息发酵程度 |
| 早盘阶段 | 9:30-10:30 | 侧重资金流向、开盘强度 |
| 盘中阶段 | 10:30-14:00 | 侧重持续性验证、板块联动 |
| 收盘阶段 | 14:00-15:00 | 侧重情绪修复、次日预期 |

### 各阶段权重调整
```markdown
盘前阶段：
  - 消息面驱动：50%
  - 市场预期：30%
  - 舆情热度：20%

早盘阶段：
  - 消息面驱动：40%
  - 市场预期：35%
  - 舆情热度：25%

盘中阶段：
  - 消息面驱动：35%
  - 市场预期：40%
  - 舆情热度：25%

收盘阶段：
  - 消息面驱动：30%
  - 市场预期：45%
  - 舆情热度：25%
```

## 态势评级标准

### 评级分类与条件

| 评级 | 符号 | 触发条件 |
|------|------|----------|
| 核爆级 | ⚡⚡⚡⚡⚡ | 多重驱动+国内外联合共振+情绪到位+催化剂到位 |
| 沸腾级 | ⚡⚡⚡⚡ | 多重驱动+情绪到位+催化剂到位 |
| 发酵级 | ⚡⚡⚡ | 单驱动+情绪到位 |
| 观望级 | ⚡⚡ | 单驱动+情绪未到位 |
| 高风险预警级 | ⚠️ | 存在3个及以上风险信号 |

### 关键概念定义

**多重驱动**：政策、技术、市场、国外、情绪中至少2个驱动类型

**国内外联合共振**：有国外技术突破+国内跟进（政策/技术/市场反应）

**情绪到位**：满足上述4个条件中的3个及以上

**催化剂到位**：有强催化剂（国外技术突破/重大政策/重大技术突破/重大事件）
