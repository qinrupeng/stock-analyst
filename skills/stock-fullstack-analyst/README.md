# 📦 A股全栈分析师技能包

## 🎯 技能概述

**技能名称**：stock-fullstack-analyst  
**功能**：输入股票代码/名称 → 生成媲美专业投行的12模块HTML深度投资分析报告  
**核心能力**：Browser实时数据抓取 + MCP情绪资讯分析 + 专业投行级分析框架  
**输出格式**：响应式HTML专业报告

## 🚀 快速开始

### 1. 安装技能
```bash
# 通过OpenClaw安装
openclaw skills install stock-fullstack-analyst
```

### 2. 激活使用
```
发送指令：请分析【股票名称 代码】
示例：
  "请分析长电科技(600584)"
  "生成北方华创(002371)的详细投资分析报告"
```

### 3. 获取报告
系统将自动：
1. 通过Browser抓取实时股价、财务、资金流数据
2. 通过MCP搜索热点资讯与舆情分析
3. 生成12模块专业HTML投资分析报告

---

## 📊 报告12大模块

| 序号 | 模块 | 内容概要 |
|:---:|------|---------|
| 1 | **报告头信息** | 股价、市值、PE/PB仪表盘 + 数据来源 |
| 2 | **执行摘要** | 综合评级、目标价、核心投资逻辑 |
| 3 | **行业深度分析** | 市场规模、竞争格局、驱动因素、风险 |
| 4 | **公司基本面** | 业务结构、财务健康、SWOT分析 |
| 5 | **财务数据详析** | 三年对比、现金流、资产负债 |
| 6 | **估值分析** | PE/PB/PS/PEG、DCF估值、同业对比 |
| 7 | **技术分析** | 趋势判断、MACD/RSI、支撑阻力位 |
| 8 | **情绪与资金面** | 舆情热度、热点新闻、主力/北向资金 |
| 9 | **ESG分析** | E/S/G评分与详细说明 |
| 10 | **情景分析** | 乐观/中性/悲观情景、敏感性分析 |
| 11 | **投资策略** | 短/中/长期操作建议、仓位控制 |
| 12 | **风险提示** | 多维风险预警与应对建议 |

## 🔧 技术架构

### 数据获取双引擎

#### Browser实时抓取
- **东方财富**：实时股价、涨跌幅、成交量
- **同花顺**：财务指标(PE/PB/ROE)、公司基本面
- **通达信网页**：技术指标(MACD/RSI/MA)
- **东方财富**：资金流向(主力/北向)

#### MCP工具
- **舆情分析**：热点资讯搜索、情绪倾向判断
- **新闻聚合**：正负面新闻分类汇总
- **行业动态**：政策/技术/需求变化分析

### 数据标注规范
```html
<span class="source">Browser抓取·东方财富·更新于14:30</span>
<span class="source">MCP新闻搜索·{关键词}·更新于14:35</span>
<span class="source">用户提供</span>
<span class="source">基于行业估算</span>
```

## 📈 核心分析框架

### 估值体系
- **相对估值**：PE/PB/PS/PEG vs 行业平均
- **绝对估值**：DCF模型（增长率、WACC、永续增长率假设）
- **同业对比**：可比公司估值表格
- **敏感性分析**：关键变量对估值影响

### 技术分析
- **趋势判断**：MA均线排列、趋势通道
- **动能指标**：MACD、RSI、KDJ
- **资金指标**：OBV、成交量、资金流向
- **波动分析**：布林带、ATR

### 情绪分析（MCP）
- **舆情热度**：0-100分值
- **热点新闻**：正面/负面/中性分类
- **资金情绪**：主力/北向流向分析
