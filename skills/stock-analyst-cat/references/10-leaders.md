## 龙头识别机制（量化时代升级版）

### 龙头定义
板块中具有**最强基本面支撑**、**最大资金关注度**、**最明确上涨逻辑**、**最优情绪资金共振**的领涨标的。

### 龙头识别标准（量化时代四维评分）

#### 基本面维度（15分）
| 标准 | 分值 | 说明 |
|------|------|------|
| 业绩支撑 | 5分 | 业绩增长确定性高 |
| 行业地位 | 4分 | 行业龙头或细分领域第一 |
| 资源禀赋 | 3分 | 拥有核心资源或技术 |
| 成长性 | 3分 | 未来增长空间大 |

> **权重调整说明**：量化时代基本面权重从35%降至15%，情绪和资金权重大幅提升

#### 市场维度（35分）
| 标准 | 分值 | 说明 |
|------|------|------|
| **资金关注度** | **10分** | 主力资金持续流入（含量化资金识别） |
| **情绪热度** | **10分** | 舆情热度、社交媒体讨论热度 |
| 涨幅领先 | 8分 | 领涨板块，涨幅靠前 |
| 成交量 | 7分 | 成交量放大，资金活跃 |

> **新增**：情绪热度指标（调用第2.6层情绪算法扫描）

#### 资金算法维度（35分）【2026-03-23新增】
| 标准 | 分值 | 说明 |
|------|------|------|
| **主力资金流向** | **12分** | 主力净流入金额和持续性 |
| **资金类型识别** | **8分** | 游资/量化/外资/机构主导 |
| **资金行为识别** | **8分** | 建仓/加仓/减仓/出货/对倒 |
| **量化资金识别** | **7分** | 量化资金流入/流出/中性 |

> **核心新增维度**：量化时代资金算法占权重35%，必须调用第2.7层资金算法扫描

#### 逻辑维度（15分）
| 标准 | 分值 | 说明 |
|------|------|------|
| 逻辑清晰 | 5分 | 上涨逻辑最直接、最明确 |
| 不可替代 | 4分 | 逻辑难以被替代或复制 |
| 预期差大 | 3分 | 市场预期差大，超预期空间大 |
| 催化剂多 | 3分 | 后续催化剂多，持续性强 |

### 龙头评分公式（量化时代）
```
龙头综合得分 = 基本面得分(15) + 市场得分(35) + 资金算法得分(35) + 逻辑得分(15)
```

> **权重说明**：情绪算法+资金算法合计占70%，体现量化时代特点

### 龙头分类（量化时代）

> **评分公式**：基本面(15) + 市场(35) + 资金算法(35) + 逻辑(15) = 100分
> **维度说明**：市场维度(35)包含情绪热度；资金算法维度(35)独立

| 龙头类型 | 特征 | 参与策略 | 量化指标 |
|----------|------|----------|----------|
| **总龙头** | 情绪+资金+逻辑三重共振 | 重仓参与，持有为主 | 总分≥75 + 市场得分≥30 + 资金得分≥30 + 逻辑得分≥12 |
| **资金龙头** | 主力资金持续流入，量化资金加持 | 跟随资金，趋势持有 | 资金算法维度≥30分 + 量化流入标识 |
| **情绪龙头** | 舆情热度高，情绪极性正面 | 快进快出，不宜恋战 | 市场维度≥30分 + 情绪传染速度>80% |
| **逻辑龙头** | 政策/技术突破，基本面硬 | 价值投资，中长期持有 | 逻辑维度≥12分 + 催化剂持续 |
| **补涨龙头** | 逻辑类似总龙头，涨幅落后 | 适度参与，博取补涨 | 总分≥50 + 市场维度≥20 + 资金维度≥15 |

> **量化时代新增**：情绪龙头和资金龙头分类，替代原有的趋势龙头

## 热点生命周期管理（量化时代升级版）

### 萌芽期
- **量化特征**：
  - 情绪热度：日增舆情<5篇，情绪得分<20
  - 资金热度：主力净流入<5亿，资金得分<15
  - 共识形成：共识形成速度指数<30%
- **传统特征**：媒体报道<5篇/日，涨停股<3只，资金净流入<10亿元
- **策略**：关注，跟踪验证
- **操作**：小仓位试探

### 爆发期
- **量化特征**：
  - 情绪热度：日增舆情>20篇，情绪得分≥40
  - 情绪极性：正面情绪占比>60%
  - 资金热度：主力净流入>30亿，资金得分≥30
  - 量化标识：量化资金流入>20%
- **传统特征**：媒体报道>20篇/日，涨停股>10只，资金净流入>50亿元
- **策略**：积极参与，抓住主升
- **操作**：重仓参与，趋势持有

### 成熟期
- **量化特征**：
  - 情绪热度：情绪得分稳定在35-45，情绪波动率>30%
  - 资金热度：资金流入放缓，资金得分25-30
  - 共识形成：共识形成速度指数50-70%（共识过度风险）
  - 情绪异常：情绪极性极度分化（正面>70% 或 负面>70%）
- **传统特征**：涨停股数量稳定，资金流入放缓，板块涨幅>15%
- **策略**：谨慎参与，关注补涨
- **操作**：减仓止盈，关注补涨

### 衰退期
- **量化特征**：
  - 情绪热度：情绪得分<20，情绪传染速度骤降
  - 情绪极性：负面情绪占比>50%
  - 资金热度：主力净流出>10亿，资金得分<15
  - 量化标识：量化资金流出标识
- **传统特征**：资金净流出，跌停股>3只，负面消息增多
- **策略**：观望为主，回避风险
- **操作**：清仓离场，回避风险

### 衰退期预警信号（进入衰退期前的提前预警）
| 预警信号 | 量化指标 | 应对措施 |
|----------|----------|----------|
| 情绪退潮 | 情绪得分日降≥15分 | 减仓30%，密切监控 |
| 资金出逃迹象 | 主力净流出≥10亿/日 | 减仓50% |
| 量化踩踏前兆 | 单只股票量化流出标识出现 | 提高警惕，跟踪观察 |
| 共识形成放缓 | 共识形成速度指数降至<30% | 关注板块轮动信号 |

> **说明**：预警信号是衰退期的前兆，当预警信号出现时应提高警惕但不必立即清仓；当量化特征全部满足时确认进入衰退期

## 涨停次日战法规则（2026-03-25复盘整合）

> **核心原则**：涨停次日默认为"风险观察日"而非"买入日"，历史经验显示追高成功率较低

### 【涨停次日风险规则】
1. 涨停次日不推荐追高（历史经验显示成功率较低）
2. 若需参与，观察开盘前30分钟走势
3. 放量低开>3%立即止损
4. 缩量高开回落是卖出信号
5. 涨停次日默认为"风险观察日"而非"买入日"

### 【涨停次日战法规则】
1. 涨停次日默认为"风险观察日"
2. 不建议涨停次日追高（历史成功率<40%）
3. 若需参与，仓位上限5%
4. 观察开盘前30分钟走势：
   - 缩量高开：可少量参与（<3%仓位）
   - 放量低开>3%：立即止损
   - 放量高开：观望为主

### 【止损位设定】
| 类型 | 止损位置 | 说明 |
|------|----------|------|
| 保守派 | 涨停次日开盘价-5% | 绝对止损 |
| 激进派 | 当日最低价-3% | 技术止损 |

### 【案例教训：航天发展（000547）】
- 3月24日涨停 → 次日不应追高
- 3月25日实际走势：大跌-2.36%
- 教训：涨停次日风险远大于机会，历史成功率<40%
