## 📋 核心功能模块

### 🎭 模块一：情绪周期量化监控

#### 情绪阶段判断逻辑：
```
情绪阶段判断函数：
输入：板块数据{涨停数量, 涨幅%, 换手率%, 资金流向}
输出：{阶段, 操作建议, 风险等级, 特征}

逻辑：
if 涨停数≥5 and 涨幅>5% and 换手率>10%:
    返回 {"🔥 高潮期", "⚠️ 准备减仓", "高", "情绪过热，随时可能退潮"}
elif 涨停数≥3 and 资金流向=="持续流入":
    返回 {"📈 发酵期", "✅ 积极介入", "中", "资金加速，行情深化"}

    返回 {"🚀 启动期", "✅ 重点关注", "低", "行情启动，聪明资金布局"}
else:
    返回 {"🥶 退潮期", "❌ 离场观望", "极高", "资金流出，情绪降温"}
```

### 🛡️ 模块三：风险扫描仪

#### 风险自检函数：
```
风险自检函数(股票代码):
    风险项 = [
        {"检查": "是否为ST/*ST", "权重": 30, "建议": "❌ 坚决回避"},
        {"检查": "流通盘>50亿", "权重": 15, "建议": "⚠️ 谨慎参与"},
        {"检查": "近3日换手率<5%", "权重": 15, "建议": "⚠️ 关注活跃度"},
        {"检查": "有退市风险公告", "权重": 30, "建议": "❌ 坚决回避"},
        {"检查": "处于利好兑现阶段", "权重": 20, "建议": "⚠️ 谨慎追高"},
        {"检查": "近10日有连续跌停", "权重": 20, "建议": "⚠️ 关注流动性"}
    ]
    
    初始分 = 100
    扣分项 = []
    
    for 项 in 风险项:
        if 触发风险(项["检查"]):
            初始分 -= 项["权重"]
            扣分项.append(f"{项['建议']} {项['检查']}")
    
    风险分 = max(初始分, 0)
    
    if 风险分 >= 80:
        等级 = "🟢 低风险"
        操作 = "✅ 可参与"
    elif 风险分 >= 60:
        等级 = "🟡 中风险" 
        操作 = "⚠️ 谨慎参与"
    else:
        等级 = "🔴 高风险"
        操作 = "❌ 坚决回避"
    
    return {分数: 风险分, 等级: 等级, 扣分项: 扣分项, 建议: 操作}
```

```

```

#### 步骤2：主力资金初筛
```

```

#### 步骤3：深度分析
```
对每只初选股票运行：
1. 🛡️ 风险扫描仪 → 过滤高风险标的
2. 🔄 轮动定位器 → 识别龙头/跟涨股
3. ⚖️ 动态评估器 → 三维评分（市场阶段自适应）
4. 🏦 龙虎榜解析器 → 分析席位行为
输出：精选股票池（5-8只）
```

#### 步骤4：交易计划制定
```
对每只精选股票：
1. 确定买入价格区间
2. 设置止损点（-5%）
3. 设置止盈点（+10-15%）
4. 确定仓位（≤20%）
5. 设置心理提示规则
输出：完整交易计划表
```
