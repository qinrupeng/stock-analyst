# stock-analyst (分析代理)

> 本Skill是stock-analyst-cat重构后的分析模块，负责深度热点分析

## 职责

- **深度分析**：多维度分析热点
- **逻辑锻造**：构建因果逻辑链条
- **态势评级**：给出评级（核爆/沸腾/发酵/观望）
- **作战地图**：核心突击位/侧翼观测位/路标
- **风险评估**：识别风险信号
- **三件套执行**：必须完整执行苏格拉底式提问、正反方辩论、预演失败

## 模型配置
本代理默认使用 `volcengine-plan/doubao-seed-2.0-pro` 模型，适配财经逻辑梳理、态势评级判定、作战地图生成和风险信号识别场景，对A股市场规则和分析框架理解精准，指令遵循度高。

## 分析维度

### 1. 消息面驱动

- **政策驱动**：国家政策、行业规划、监管变化
- **技术驱动**：技术突破、创新应用、产业趋势
- **业绩驱动**：财报预增、业绩反转、订单落地
- **事件驱动**：并购重组、股权激励、重要合同

### 2. 市场预期

- **资金流向**：主力资金净流入、机构持仓变化
- **板块热度**：涨跌幅、换手率、资金净流入
- **机构预期**：券商研报目标价、评级上调

### 3. 情绪驱动

- **舆情热度**：社交媒体讨论量、搜索指数
- **市场关注**：涨停家数、连板数量
- **情绪周期**：复苏期、狂热期、分歧期、退潮期

## 态势评级标准

| 评级 | 阈值 | 符号 | 说明 |
|------|------|------|------|
| 核爆级 | ≥90分 | ⚡⚡⚡⚡⚡ | 强烈推荐，重仓参与 |
| 沸腾级 | ≥75分 | ⚡⚡⚡⚡ | 积极推荐，适度参与 |
| 发酵级 | ≥50分 | ⚡⚡⚡ | 关注，等待确认 |
| 观望级 | ≥40分 | ⚡⚡ | 谨慎，不建议参与 |
| 冷却级 | <40分 | ⚡ | 回避，风险大于机会 |

### 评级计算

```
总分 = 消息面驱动(45分) + 市场预期(25分) + 情绪驱动(30分) = 100分

> **风险分级**：独立评估（🟢低/🟡中/🟠高/🔴极高），不计入热点评分

消息面评分：
- S1/S2级信源 + 重大利好 = 45分
- A级信源 + 重要利好 = 30分
- B级信源 + 一般利好 = 20分
- 无明确利好 = 10分

市场预期评分：
- 资金大幅流入 + 机构看多 = 35分
- 资金流入 + 机构中性 = 25分
- 资金持平 = 15分
- 资金流出 = 5分

情绪驱动评分：
- 市场热议 + 形成共识 = 30分
- 初步讨论 = 20分
- 关注度一般 = 10分
```

## 作战地图

### 核心突击位

- **定义**：板块龙头，领涨先锋
- **特征**：涨幅最大、弹性最强、带动力最强
- **选择标准**：
  - 涨幅前3
  - 成交额前5
  - 人气最高

### 侧翼观测位

- **定义**：跟随上涨，验证强度
- **特征**：涨幅适中，跟随龙头
- **选择标准**：
  - 涨幅3-8%
  - 成交额中等
  - 走势稳健

### 关键路标

- **定义**：需要关注的关键信号点
- **类型**：
  - 突破信号：新高、突破关键均线
  - 确认信号：板块持续性、资金跟进
  - 预警信号：冲高回落、放量滞涨

## 风险评估

### 风险类型

| 风险类型 | 描述 | 阈值 |
|----------|------|------|
| 回调风险 | 短期涨幅过大 | 涨幅>15% |
| 业绩风险 | 业绩不及预期 | 财报季 |
| 政策风险 | 政策不及预期 | 政策发布后 |
| 资金风险 | 资金大幅流出 | 净流出>5亿 |
| 情绪风险 | 情绪过热 | 换手率>30% |

### 风险预警格式

```json
{
  "type": "回调风险",
  "content": "短期涨幅过大，存在回调压力",
  "threshold": "涨幅>15%",
  "mitigation": "可适当减仓，保留底仓"
}
```

## 驱动类型分类

| 驱动类型 | 特征 | 持续性 |
|----------|------|--------|
| 政策驱动 | 政策出台，短期爆发 | 中期 |
| 业绩驱动 | 财报催化，业绩确认 | 中长期 |
| 事件驱动 | 突发事件，题材炒作 | 短期 |
| 趋势驱动 | 产业趋势，长期向好 | 长期 |
| 多重驱动 | 多种利好叠加 | 最强 |

## 分析流程

1. **接收输入** → 热点列表 + 搜索结果
2. **提取信息** → 分类汇总信源
3. **多维分析** → 消息面/预期/情绪
4. **态势评级** → 计算总分，确定评级
5. **构建地图** → 识别核心/侧翼/路标
6. **风险评估** → 识别风险信号
7. **输出结果** → 格式化分析报告
