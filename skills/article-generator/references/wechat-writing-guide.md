# 微信公众号写作风格指导（适配私域属性）

## 概览
本文档提供微信公众号平台的文章写作规范与技巧，帮助创作者生成符合公众号调性和私域用户喜好的优质内容，兼顾深度、可信度和传播性。

## 公众号平台特点（与头条的核心区别）
- **用户心智**：主动关注，深度阅读，信任度高，属于私域流量，用户愿意花更多时间读长文
- **内容生命周期**：永久沉淀，可反复传播，长尾效应强，不像头条只有24小时热度
- **交互模式**：点赞/在看/收藏/星标/打赏/私域导流，核心目标是沉淀用户，建立长期信任
- **内容偏好**：拒绝短平快的碎片化内容，更喜欢有深度、有逻辑、有个人观点的干货内容
- **传播逻辑**：社交裂变，用户觉得有价值才会分享到朋友圈/好友，传播性取决于内容质量，而非算法推荐

## 标题创作技巧（公众号专属）

### 标题原则
- **字数控制**：20-28字最佳，最多不超过32字，完整显示在公众号会话列表
- **价值前置**：一眼看出读完能获得什么，减少用户决策成本
- **情绪共鸣**：戳中目标用户的痛点/爽点/焦虑点
- **避免标题党**：公众号用户反感夸张标题，更信任真实、有价值的标题
- **关键词前置**：重要信息放在标题前半部分，方便用户快速抓取

### 公众号爆款标题类型（适配财经内容）
1. **痛点型**："A股踩坑无数？这3个避坑技巧让你少亏30%"
2. **利益型**："算电协同风口已来，提前布局这2个板块坐等起飞"
3. **悬念型**："华为昇腾大会今天开幕，我预判了主力的预判"
4. **数字型**："2026年最确定的3条投资主线，现在买刚好"
5. **反差型**："主力都在买的算力板块，为什么我劝你别追高？"

### 标题禁忌
- ❌ 夸张震惊体："震惊！A股要暴跌了"
- ❌ 极端绝对化："明天这只股100%涨停"
- ❌ 过于直白："3月19日A股盘前分析"（太干，没有点击欲望）
- ❌ 蹭无关热点：标题带和内容无关的热门关键词

## 开头写法（公众号专属）

### 开头原则
- **建立信任**：开头用1-2句话拉近和读者的距离，像和朋友聊天一样
- **痛点共鸣**：直接点出读者最关心的问题，让他觉得"这篇就是写给我的"
- **预告价值**：告诉读者读完这篇能获得什么，值不值得花时间读
- **控制字数**：150-200字，比头条开头稍长，建立熟悉感

### 公众号开头专属技巧（适配财经内容）
#### 1. 共情切入
"最近是不是看着AI板块涨上天，自己手里的票却一动不动？怕追高被套，不追又怕错过行情，是不是特别纠结？今天这篇就给你把算力板块的逻辑说透，看完你就知道该怎么操作了。"

#### 2. 数据切入
"今天主力资金净流入电力板块13.17亿，金开新能、广安爱众直接涨停。很多人还不知道，这背后是国家刚刚推出的算电协同新基建政策，万亿级的市场刚启动。"

#### 3. 个人观点切入
"我研究A股这么多年，很少见到政策+技术+资金三重共振的板块，今天的算电协同就是其中之一。接下来我给你拆解一下这个板块的逻辑和机会。"

#### 4. 热点切入
"华为合作伙伴大会今天开幕，英伟达GTC大会刚发了新芯片，两件大事撞到一起，算力板块接下来的走势已经很明确了。"

### 开头禁忌
- ❌ 套话空话："大家好，今天我给大家分析一下今天的A股行情"
- ❌ 过于正式："本文将从政策、资金、技术三个维度分析算电协同板块的投资价值"
- ❌ 太长：开头超过300字，读者直接划走

### 开头排版规范（必须遵守！）
**三段式结构**（手机阅读友好）：
```
段1（痛点）：你是不是也遇到过这种情况...（疑问句引发共鸣）
段2（共情+核心观点）：其实这种纠结我特别理解。**A股的机会从来不是普涨，而是结构性行情。**（核心观点黑色加粗）
段3（价值预告）：**今天这篇，我就给你把当前最确定的算力主线逻辑说透，让你知道接下来该怎么操作。**（价值预告蓝色加粗）
```

**要点**：
- 每段控制在80字以内
- 核心观点用黑色加粗：传达文章核心判断
- 价值预告用蓝色加粗 `<strong style="color: #0F4C81;">`：告诉读者读完能获得什么

## 正文结构（公众号专属，适配A股内容）

### 标准结构（私域友好型，1500-2500字）
```
标题（20-28字）
├── 开头（150-200字：共情+痛点+价值预告）
├── 核心逻辑拆解（400-600字）
│   └── 政策/事件背景 + 核心逻辑 + 市场空间
├── 热点深度分析（每个热点300-400字，共2-3个热点）
│   ├── 热点1：逻辑 + 龙头标的 + 持续性判断
│   ├── 热点2：逻辑 + 龙头标的 + 持续性判断
│   ├── 热点3：逻辑 + 龙头标的 + 持续性判断
├── 风险提示（150-200字，强制要求）
│   └── 明确说明风险点，不做确定性承诺
├── 主线观察（150-200字）
│   └── 各主线当前所处窗口（确认/黄金/观察），不提供买卖区间/仓位/止损位等具体操作指引
└── 结尾（200-300字：总结+互动+关注引导）
```

### 章节写作要点
#### 逻辑递进
每个章节层层深入，不要零散罗列热点：
- 先讲大背景 → 再讲具体机会 → 最后讲风险和操作
- 每章结尾用1句话承上启下："讲完了政策逻辑，我们再看看资金面的表现"

#### 段落控制
- 每段80-120字，比头条段落稍短，手机阅读更舒服
- 多用短句，避免长难句
- 重点内容加粗，方便用户快速抓取核心信息

#### 观点呈现（财经内容专属）
```
核心观点（加粗，1句）
├── 逻辑解释（1-2句，通俗易懂）
├── 数据支撑（带来源，比如"据东方财富数据，今日电力板块主力净流入13.17亿"）
├── 落地建议（1句，比如"这个板块短期可以关注，但不要追高"）
```

## 语言风格（公众号专属）

### 核心特点
- **人设统一**：保持专业严谨又不失温度的人设风格，让读者觉得是一个靠谱的老朋友在跟他聊天
- **有温度**：避免冷冰冰的数据分析，加入一点个人观点和判断
- **专业严谨**：财经内容不能出错，所有数据要有来源，所有判断要留有余地
- **通俗易懂**：把专业术语用大白话讲出来，不要卖弄专业知识

### 语言技巧
#### 1. 保持风格一致性
多用固定的表达习惯，比如：
- "核心判断："
- "持续性判断："
- "给你提个醒："
- "我的判断是："

#### 2. 多用第二人称"你"
"你是不是也遇到过这种情况？"
"给你的建议是："
"你可以重点关注这几个标的"

#### 3. 避免绝对化表述
❌ 错误："这个板块明天肯定涨"
✅ 正确："这个板块短期上涨概率比较大，但要注意XX风险"

#### 4. 专业术语解释
遇到专业术语要简单解释，比如：
"算电协同，简单说就是把数据中心建在绿电厂旁边，直接用绿电供电，成本比用电网电低30%以上。"

### 避免的表达
- ❌ 学术化表达："基于上述分析，我们可以得出如下结论"
- ❌ 官腔套话："综上所述，我们认为该板块具有较高的投资价值"
- ❌ 模棱两可："可能大概也许会涨"（要有明确的观点，但要说明风险）
- ❌ 极端表述："100%涨停"、"稳赚不赔"

## 数据与案例使用（公众号专属）

### 数据使用原则
- **数据要准**：所有数据必须有明确来源，比如"东方财富Choice数据"、"国家能源局"、"公司公告"
- **数据要新**：优先使用T+1以内的最新数据，不要用超过72小时的旧数据
- **标注来源**：重要数据必须标注来源，提升可信度
- **适量使用**：一篇文章3-5个核心数据即可，不要堆砌数据

### 案例使用原则
- **用近期案例**：最好用1个月以内的市场案例，用户有感知
- **用真实案例**：不要编造案例，比如"昨天金开新能涨停"就比"某电力股涨停"可信度高
- **用散户熟悉的案例**：多用散户关注度高的龙头股，少用冷门标的

## 结尾技巧（公众号核心，私域转化关键）

### 结尾原则
- **总结到位**：用1-2句话总结全文核心观点，方便用户记住
- **互动引导**：提出开放性问题，鼓励用户留言互动，提升账号活跃度
- **关注引导**：自然引导用户关注、星标，沉淀私域用户
- **风险提示**：再次提醒投资有风险，符合合规要求
- **控制字数**：200-300字

### 公众号专属结尾模板（财经内容通用）
```
总结一下，今天的核心机会是算电协同、华为昇腾和光模块三个方向，其中算电协同的确定性最高。
最后给大家提个醒，现在大盘整体偏弱，操作上一定要控制仓位，不要追高，再好的逻辑也要买在低点。
你今天看好哪个方向？欢迎在评论区聊聊~
如果觉得这篇对你有帮助，记得 👍点赞 + 👁️在看 + ⭐星标，下次有好机会我第一时间告诉你。
投资有风险，入市需谨慎，本文仅代表个人观点，不构成投资建议。
```

### 结尾禁忌
- ❌ 没有互动引导：写完直接结束，不给用户留言的理由
- ❌ 硬广推销：结尾硬推课程/星球，引起用户反感
- ❌ 没有风险提示：财经内容必须有明确的风险提示，避免合规风险

## 排版建议（公众号专属）

### 文字排版
- **段落间距**：每段之间空一行，不要堆在一起
- **重点突出**：核心观点、重要数据、风险提示用加粗标注
- **小标题**：每个章节用清晰的小标题，比如"一、算电协同核心逻辑"
- **列表使用**：并列信息用有序/无序列表，清晰易读
- **字体大小**：默认字号即可，不要用太小的字体

### 结构化元素
- **风险提示框**：风险提示用灰色背景或引用格式标出，醒目
- **数据表格**：资金流向、板块涨跌幅用表格呈现，清晰直观
- **分隔线**：章节之间用细分隔线分开，结构更清晰

### HTML排版规范（必须遵守！）
**引用框间距统一**：`margin: 20px 0`

**数据卡片**：
- 统一三列布局，每列 `width: 33%`
- 卡片背景用浅蓝色 `#E3F2FD`
- 数字用大字号（24-28px）+ 蓝色加粗

**引用框类型**：
- 蓝色渐变（💡核心判断）：`linear-gradient(135deg, #E3F2FD 0%, #BBDEFB 100%)`，左边框 `#0F4C81`
- 绿色渐变（✅持续性判断）：`linear-gradient(135deg, #E8F5E9 0%, #C8E6C9 100%)`，左边框 `#43A047`
- 橙色渐变（⚠️注意）：`linear-gradient(135deg, #FFF3E0 0%, #FFE0B2 100%)`，左边框 `#FF9800`
- 红色渐变（⚠️风险提示）：`linear-gradient(135deg, #FFEBEE 0%, #FFCDD2 100%)`，左边框 `#EF5350`
- 灰色（💡简单解释）：`background: #f8f9fa`，左边框 `#78909C`

**股票代码**：用浅灰色 `<span style="color: #999;">（600396）</span>` 弱化，减少视觉噪音

**结尾规范**：
```
┌─────────────────────────────────┐
│ 💡 投资有风险，谨慎前行          │  ← 蓝色渐变框
│ 方向对了，节奏也很重要...        │
│ 本文仅代表个人观点...            │
└─────────────────────────────────┘

   你觉得哪个方向最有确定性？
      欢迎在评论区聊聊～

   觉得有用？记得 点赞 + 在看 + 星标
```
- 不要加"收藏备用"框，显得多余
- 互动引导居中显示

### 配图排版
- **封面图**：2.35:1比例，符合公众号封面尺寸要求，清晰突出主题
- **内文配图**：每2-3个章节配1张图，风格统一，和内容相关
- **图片说明**：每张图配1-2句话的说明，解释图片内容
- **避免水印**：所有图片不要带第三方水印，保持整洁

## 合规要求（财经内容专属，强制遵守）
1. **必须加风险提示**：文章末尾必须明确标注"投资有风险，入市需谨慎，本文仅代表个人观点，不构成投资建议"
2. **禁止具体操作建议**：不能给出"明天买入XX股票"、"XX股票目标价XX元"这类具体买卖建议，只能给方向性参考
3. **禁止确定性承诺**：不能说"肯定涨"、"必赚"、"100%盈利"这类绝对化表述
4. **数据必须标注来源**：重要数据要说明来源，比如"据东方财富数据"
5. **禁止蹭敏感热点**：不要谈论政策敏感内容，不要传播不实信息

## 写作检查清单（公众号专属）
生成文章后，按以下清单检查：
### 内容检查
- [ ] 标题是否有价值点，能吸引目标用户点击
- [ ] 开头是否有共情，能抓住用户注意力
- [ ] 核心逻辑是否清晰，层层递进
- [ ] 每个观点是否有数据/案例支撑
- [ ] 风险提示是否明确，符合合规要求
- [ ] 结尾是否有互动和关注引导（👍点赞 + 👁️在看 + ⭐星标带emoji）
### 语言检查
- [ ] 是否保持了统一的风格（不能出现"喵娘"等特定人设名称）
- [ ] 语言是否通俗易懂，没有晦涩术语
- [ ] 是否避免了绝对化表述
- [ ] 有没有错别字和语病
- [ ] **禁止股票代码**（6位数字如002361/600938，只能用股票名称）
### 结构检查
- [ ] 总字数在1500-2500字之间
- [ ] 段落长度适中，手机阅读友好
- [ ] 重点内容都有加粗突出
- [ ] 结构完整，逻辑通顺
### 平台适配
- [ ] 符合公众号私域调性，适合长期沉淀
- [ ] 有足够的干货内容，值得用户收藏分享
- [ ] 配图尺寸符合公众号要求
- [ ] 合规要求全部满足
