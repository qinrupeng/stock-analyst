# 实施时间表与执行流程

本文档定义喵娘分析技能的实施时间表和执行流程。

---

## 实施时间表

### 周一盘前（8:00-9:00）

#### 执行顺序
1. **8:00-8:15** - 第0层市场全景扫描
   - 快速定位当日所有潜在热点
   - 搜索主流财经媒体、交易所公告
   - 输出：潜在热点清单

2. **8:15-8:30** - 第1层官方确认层
   - 验证热点的官方/权威信息源
   - 搜索国务院、证监会、发改委等官网
   - 输出：经过官方验证的热点清单

3. **8:30-8:45** - 第2层专业解读层
   - 获取券商研报、专业媒体分析
   - 理解热点的深层逻辑和影响
   - 输出：带有深度逻辑的热点清单

4. **8:45-9:00** - 建立逻辑链条与评分
   - 串联各层信息，建立因果关系
   - 执行验证流程，评估舆情热度
   - 综合评分并输出预警

### 周一早盘（9:30-10:00）

#### 执行顺序
1. **9:30-9:40** - 实时监控开盘异动
   - 观察开盘竞价情况
   - 识别开盘异动板块
   - 验证盘前预测准确性

2. **9:40-9:50** - 调整评分权重
   - 根据开盘情况调整评分
   - 更新热点优先级
   - 修正预测结论

3. **9:50-10:00** - 输出热点确认报告
   - 确认热点有效性
   - 更新态势评级
   - 输出确认报告

### 盘中持续（10:00-14:00）

#### 执行顺序
1. **10:00-11:30** - 跟踪热点演变
   - 监控热点板块走势
   - 捕捉新驱动因素
   - 验证逻辑持续性

2. **11:30-13:00** - 午盘休整
   - 复盘上午走势
   - 准备午后策略
   - 识别午后机会

3. **13:00-14:00** - 下午监控
   - 跟踪午后热点演变
   - 实时风险排查
   - 动态调整策略

### 收盘复盘（14:00-15:00）

#### 执行顺序
1. **14:00-14:30** - 复盘全天走势
   - 复盘全天热点走势
   - 验证预测准确性
   - 识别预测偏差

2. **14:30-14:45** - 识别次日机会
   - 识别次日潜在热点
   - 建立热点观察清单
   - 准备次日策略

3. **14:45-15:00** - 输出复盘报告
   - 输出复盘报告
   - 总结经验教训
   - 优化次日策略

## 实时监控与动态调整

### 盘中监控要点

#### 1. 热点演变监控
- 热点板块涨幅变化
- 涨停股数量变化
- 资金流向变化
- 板块联动效应

#### 2. 新驱动因素捕捉
- 突发消息
- 政策变化
- 业绩公告
- 资金异动

#### 3. 风险信号监控
- 资金出逃信号
- 情绪降温信号
- 逻辑证伪信号
- 监管风险信号

### 动态调整策略

#### 评分调整
- 根据实时信息调整评分
- 更新热点优先级
- 修正预测结论

#### 评级调整
- 根据最新情况更新评级
- 调整操作建议
- 更新风险提示

#### 策略调整
- 根据市场变化调整策略
- 修正作战地图
- 更新操作计划

## 智能日期识别机制

### 日期识别逻辑
```python
def 获取搜索日期():
    today = datetime.now()
    weekday = today.weekday()
    
    if weekday == 0:  # 周一
        # 搜索上周五 + 周末两天
        return {
            "主时间窗口": (today - timedelta(days=3)).strftime("%Y-%m-%d"),  # 上周五
            "补充时间窗口": [
                (today - timedelta(days=2)).strftime("%Y-%m-%d"),  # 周六
                (today - timedelta(days=1)).strftime("%Y-%m-%d"),  # 周日
            ],
            "实时时间窗口": today.strftime("%Y-%m-%d")  # 今日
        }
    else:
        # 搜索昨天 + 今日
        return {
            "主时间窗口": (today - timedelta(days=1)).strftime("%Y-%m-%d"),  # 昨天
            "补充时间窗口": [],
            "实时时间窗口": today.strftime("%Y-%m-%d")  # 今日
        }
```

### 时间窗口应用

| 场景 | 主时间窗口 | 补充时间窗口 | 实时时间窗口 |
|------|------------|--------------|--------------|
| 周一盘前 | 上周五+周末 | - | 今日 |
| 平日盘前 | 昨日 | - | 今日 |
| 盘中 | 近日 | 近日+实时 | 实时 |
| 盘后 | 今日 | 近日 | - |
