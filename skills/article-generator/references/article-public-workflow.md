# 通用文章生成公共流程

> 所有平台（头条/公众号/小红书等）共享的流程框架

---

## 10步执行框架（所有平台通用）

```
□ 1. 确认需求 → 用户说清楚则确认，不清楚则追问
□ 2. 读取参考文档 → 根据平台读取对应writing-guide
□ 3. 定策略 → 选题角度、标题方向、内容结构
□ 4. 爆款分析 → 搜索近期爆款参考（可跳过）
□ 5. 素材补充 → 搜索主题相关资讯
□ 6. 确认大纲 → 用户确认后再写
□ 7. 写文章 → 按平台writing-guide规则
□ 8. 配图生成 → 根据类型选风格
□ 9. 质量检查 → 完整性+写作质量+平台适配
□ 10. 输出生成 → 交付给用户
```

---

## 第1步：确认需求

### 场景判断

| 场景 | 操作 |
|------|------|
| "根据热点报告写头条/公众号" | 确认主题清晰 → 确认报告路径 → 进入下一步 |
| "写一篇关于xxx的文章" | 追问：类型/字数/调性 → 进入下一步 |
| "帮我写头条/公众号" | 追问：主题/类型/字数/调性 |

### 需求明确时（直接确认）
```
✓ 主题：xxx
✓ 平台：xxx
✓ 字数：xxx
确认无误，进入下一步
```

---

## 第3步：定策略（单篇文章微观策略）

- 选题角度：这篇文章要讲什么？
- 标题方向：提问型/数字型/对比型/利益型？
- 内容结构：分几个章节？
- 核心观点：读者能获得什么？

**📌 content-strategy**：需要系统性规划时读取 `../content-strategy/SKILL.md`

---

## 第4步：爆款分析

**📌 social-content-generator-0.1.0**：读取 `../social-content-generator-0.1.0/SKILL.md`

核心获取：
- 爆款逆向工程6步框架（NICHE→SCRAPE→ANALYZE→PLAYBOOK→LAYER→CONVERT）
- 钩子公式（好奇心/故事/价值/反直觉/社会证明）

### 分析结果呈现

```
爆款文章分析结果：
- 标题风格：[类型]，平均X字
- 文章结构：X个章节，总字数X字
- 互动钩子：X个，分布在开头/章节结尾/总结
- 配图风格：[风格]
- 钩子模式：[具体模式]
- 可复用模板：[模板]

是否采纳这些写作技巧？（A. 采纳 / B. 不采纳）
```

## 第5步：素材补充

**📌 seo-content-writer-2.0.0 + seo-1.0.3**：读取对应SKILL.md

- 关键词研究流程（主关键词 + 长尾关键词）

1. <主题> 最新动态

4. <主题> 专家观点

### 输出示例

```
📋 素材补充结果：
🔍 主关键词：[词]（搜索量X，难度Y）
🔍 长尾关键词：[词1, 词2, 词3]
```

## 第6步：确认大纲

```
标题：<暂定标题>

1. <章节1标题> - <核心观点>
2. <章节2标题> - <核心观点>
3. <章节3标题> - <核心观点>
4. 总结 - <总结要点>

请确认：
A. 大纲OK，开始生成文章
B. 需要调整（请告知具体修改意见）
```

## 第8步：配图生成

**📌 imagen + stock-report-image**

### 配图策略

| 文章类型 | 配图方式 | 配图风格 |
|----------|---------|---------|
| 科技/观点/分析类 | 文生图或混合 | 科技未来风/抽象概念风 |
| 新闻/故事/案例类 | 网络图或混合 | 写实照片风 |
| 教程/技巧类 | 文生图或网络图 | 插画卡通风 |
| 其他类型 | 混合模式 | 智能推荐 |

### 配图数量选择

```
A. 最少配置：封面图 + 1-2张章节插图
B. 标准配置：封面图 + 2-3张章节插图（推荐）
C. 丰富配置：封面图 + 3-5张章节插图
```

### 水印检查规则
- 所有网络图片必须通过水印检查
- 未通过改用文生图
- 最多3轮替代搜索

### A股股票类文章封面规则

> 读取 `../stock-report-image/SKILL.md`

🚨 **强制规则**：
- 必须用stock-report-image生成封面，禁止直接调用imagen
- 必须先读取报告内容，再匹配风格，再生成提示词
- 必须使用逐格精确提示词

---

## 第9步：质量检查

### 通用完整性检查

- [ ] 标题符合平台字数限制
- [ ] 开头快速切入主题
- [ ] 3-5个章节，结构完整
- [ ] 有互动钩子
- [ ] 字数达标
- [ ] 每个观点有数据或案例支撑
- [ ] 语言口语化，无学术化表达
- [ ] 重点内容加粗突出

### 平台专属检查

**头条**：标题≤30字 + 开头100-150字 + 结尾有互动引导
**公众号**：标题≤28字 + 共情开头 + 风险提示 + 关注引导 + HTML排版规范

**未全部打勾，禁止输出！**

### 问题修复

- 封面图缺失 → 重新生成
- 配图数量不足 → 补充生成
- 图片说明缺失 → 自动生成20-50字说明
- 水印检查不通过 → 改用文生图

## 第10步：输出生成

### 必须输出质量声明

```
✅ 标题技巧：[类型]
✅ 开头技巧：[类型]
✅ 结尾互动钩子：[类型]
✅ 语言风格：[风格]
[头条] ✅ SEO关键词：[主关键词 + 长尾关键词]
[公众号] ✅ 合规检查：[结果]
[公众号] ✅ 私域价值：[评估]
```

### 自检三问

1. 技能调用声明写了没？
2. 质量声明写了没？
3. 大爷可能问什么？

**自检没通过 = 不能输出！**

## 通用禁止事项

**【违反必罚】**：
- ❌ **禁止跳过步骤** → 必须按10步执行
- ❌ **禁止省略质量检查** → 自检清单必须逐项打勾
- ❌ **禁止出现股票代码** → 不得出现6位数字代码（如000938/300136）
- ✅ **允许出现股票名称** → 可以提"紫光股份"等名称

**【A股股票类强制规则】**：
- 🚨 必须调用stock-report-image生成封面
- 🚨 必须先读取报告内容再匹配风格

**【违反后果】**：
- 被大爷抓包 = 写3000字检讨
- 连续2次偷懒 = 自罚停止接单1天

## 错误处理

| 错误类型 | 处理方式 |
|----------|----------|
| 需求不完整 | 询问缺失信息 |
| 素材搜索失败 | 调整搜索关键词 |
