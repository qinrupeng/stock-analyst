# 态势评级

**文档版本**：1.1（2026-03-23 更新：沸腾级阈值修正为≥75分，与02-scoring.md权威定义保持一致）

## 评级体系

### 五级评级

| 评级 | 阈值 | 符号 | 推荐程度 | 仓位建议 |
|------|------|------|----------|----------|
| 核爆级 | ≥90分 | ⚡⚡⚡⚡⚡ | 强烈推荐 | 30-50% |
| 沸腾级 | ≥75分 | ⚡⚡⚡⚡ | 积极推荐 | 15-30% |
| 发酵级 | ≥50分 | ⚡⚡⚡ | 关注 | 5-15% |
| 观望级 | ≥40分 | ⚡⚡ | 谨慎 | 0-5% |
| 冷却级 | <40分 | ⚡ | 回道 | 0% |

### 评分公式

```
总分 = 消息面分 + 市场预期分 + 情绪驱动分

消息面分 = 政策分 + 技术分 + 业绩分 + 事件分
市场预期分 = 资金分 + 板块分 + 机构分
情绪驱动分 = 舆情分 + 周期分
```

## 评分细则

### 消息面评分 (满分45分)

#### 政策分 (15分)
- 国务院级重大政策：15分
- 部委级重要政策：12分
- 省级政策：8分
- 无政策：5分

#### 技术分 (10分)
- 技术突破+商用落地：10分
- 技术突破+预期商用：7分
- 技术进展：5分
- 无技术：3分

#### 业绩分 (10分)
- 业绩暴增+确定性高：10分
- 业绩增长+确定性中：7分
- 业绩预期：5分
- 无业绩：3分

#### 事件分 (5分)
- 重大事件（并购/发布）：5分
- 一般事件：3分
- 无事件：1分

#### 国外突破分 (5分)
- 国外重大技术突破+国内跟进：5分
- 国外技术突破+国内预期跟进：3分
- 无国外技术突破：0分

### 市场预期评分 (满分25分)

#### 资金分 (15分)
- 主力净流入>5亿：15分
- 主力净流入1-5亿：10分
- 主力净流入-1~1亿：5分
- 主力净流入<-1亿：2分

#### 板块分 (10分)
- 涨幅>5%+换手率>15%：10分
- 涨幅3-5%+换手率10-15%：7分
- 涨幅1-3%+换手率5-10%：4分
- 其他：2分

### 情绪驱动评分 (满分30分)

#### 舆情分 (15分)
- 热搜/热议/讨论暴增：15分
- 讨论量较高：10分
- 讨论量平稳：5分
- 无人问津：2分

#### 周期分 (10分)
- 复苏期/狂热期：10分
- 分歧期：6分
- 退潮期：3分

#### 情绪传染分 (5分)
- 情绪快速传染：5分
- 情绪缓慢传染：3分
- 无情绪传染：0分

## 评级调整规则

### 加分项

- **多重驱动叠加**：+5分
- **S级信源确认**：+5分
- **机构一致看多**：+5分
- **北向资金大幅买入**：+5分
- **首次覆盖/评级上调**：+3分

### 减分项

- **市场已充分预期**：-5分
- **机构存在分歧**：-5分
- **业绩不及预期风险**：-5分
- **政策不及预期风险**：-5分
- **技术落地不确定**：-5分
- **涨幅过大**：-10分
