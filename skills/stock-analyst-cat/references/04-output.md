# 输出格式规范

本文档定义喵娘分析技能的标准输出格式。

---

## 标准头部格式

```markdown
🕒 分析基准时刻（此刻）：[YYYY-MM-DD HH:MM:SS] (来源：{系统时钟/用户指定})
⏰ 当前阶段：{盘前阶段/早盘阶段/盘中阶段/收盘阶段/非交易时间}
📊 当前评分机制：{盘前阶段评分机制/早盘阶段评分机制/盘中阶段评分机制/收盘阶段评分机制}
🌐 信息获取方式：{联网搜索成功执行 / 联网搜索失败，启用备用知识库}
🔍 质疑状态：{已对判断进行质疑 / 未对判断进行质疑}
⚠️ 风险等级：{🟢低风险 / 🟡中风险 / 🟠高风险 / 🔴极高风险}
🔥 热点发现状态：{发现X个新热点 / 无新热点}
🌍 国内外联合共振状态：{发现X个国内外联合共振热点 / 无国内外联合共振热点}
🚀 多重驱动状态：{发现X个多重驱动热点 / 无多重驱动热点}
💡【核心发现】用一句话点明今日最关键的"状态转换"信号。
```

---

## 热点主题报告格式

### 主题命名规范
**格式**：【目标状态】+【核心驱动】+【风险等级】

**示例**：
- 【产能紧张】+【政策驱动】+【🟡中风险】
- 【技术突破】+【国外驱动】+【🟢低风险】
- 【需求爆发】+【多重驱动】+【🟠高风险】

### 态势评级格式
**格式**：【知识库标准态势评级】+【喵娘锐评：50字内精炼点评】+【风险等级：🟢/🟡/🟠/🔴】

**示例**：
- ⚡⚡⚡⚡⚡核爆级 + 【喵娘锐评：国内外技术共振+政策密集催化，龙头已5连板】 + 【风险等级：🟡中风险】

### 状态转换格式
**格式**：正从 **【旧状态】** 向 **【新状态】** 迁移。

**示例**：正从 **【产能过剩】** 向 **【供需紧平衡】** 迁移。

### 驱动类型格式
**可选值**：政策驱动、技术驱动、市场驱动、国外驱动、情绪驱动、多重驱动

## 关键拼图格式

```markdown
**🧩 关键拼图（变量）**：
*   [🟢S级] [YYYY-MM-DD]-信源名称：[信息精要] （对市场生态的冲击解读） {时效标识}
*   [🔵A级] [YYYY-MM-DD]-信源名称：[信息精要] （对市场生态的冲击解读） {时效标识}
*   [🟡B级] [YYYY-MM-DD]-信源名称：[信息精要] （对市场生态的冲击解读） {时效标识}
*   **`🔍 质疑点`**：[对自己判断的质疑，寻找反面证据]
*   **`⚠️ 风险点`**：[识别出的风险信号]
```

**要求**：
- 信源必须按 S级→A级→B级→C级 顺序排列
- 每个信源必须包含"对市场生态的冲击解读"
- 必须包含质疑点和风险点

## 作战地图格式

```markdown
**🗺️ 作战地图**：
*   **🎯 核心突击位**：`代码 名称` - 最硬核逻辑与当前博弈价值点，**风险可控**。
*   **🧩 侧翼观测位**：`代码 名称` - 观测逻辑，**需进一步验证**。
*   **🧭 关键路标**：（列出1-2个可验证的短期市场信号）
*   **🚨 风险路标**：（列出1-2个可能导致博弈失败的破局信号）
```

**要求**：
- 核心突击位必须标注"风险可控"
- 侧翼观测位必须标注"需进一步验证"
- 关键路标必须是可验证的短期市场信号
- 风险路标必须是可能导致博弈失败的破局信号

---

## 喵娘洞见格式

```markdown
**☯️ 喵娘洞见（六维推演）**：
1.  **🌱 生态位变化**：（机会本质是什么？改变了产业链哪里的价值分配？）
2.  **🎭 主力剧本推演**：（基于当前风格，主力最可能如何行动？节奏如何？）
3.  **⏰ 节点必然性**：（为何是现在？下一个关键时间节点是什么？）
4.  **⚠️ 风险边界分析**：（风险边界在哪里？什么情况下会触发风险？）
5.  **🌍 国内外联合共振分析**：（国外技术突破如何引爆情绪？国内如何跟进形成共振？增量市场在哪里？）
6.  **🚀 多重驱动分析**：（政策、技术、市场、国外、情绪五重驱动如何共振？驱动强度如何？）
```

**要求**：
- 必须包含全部六个维度
- 每个维度必须提供实质性分析
- 避免空洞的模板化表述

## 盘前预警输出格式

```markdown
🚨【盘前预警】
⚠️ 风险等级：{🟢/🟡/🟠/🔴}
🎯 预警类型：{国外技术突破/国内重大政策/外围市场/资金异常/重大利空}
📝 预警内容：（简要描述预警原因和可能影响）
💡 应对建议：（简要建议如何应对）
```

## 完整示例

```markdown
🕒 分析基准时刻（此刻）：2026-02-12 14:30:00 (来源：系统时钟)
⏰ 当前阶段：盘中阶段
📊 当前评分机制：盘中阶段评分机制
🌐 信息获取方式：联网搜索成功执行
🔍 质疑状态：已对判断进行质疑
⚠️ 风险等级：🟡中风险
🔥 热点发现状态：发现3个新热点
🌍 国内外联合共振状态：发现1个国内外联合共振热点
🚀 多重驱动状态：发现1个多重驱动热点
💡【核心发现】小金属板块因国外钨矿供给收缩+国内收储预期，叠加新能源需求爆发，态势评级上调至沸腾级。

---

### 💎 主题：【供需紧平衡】+【多重驱动】+【🟡中风险】
**态势评级**：⚡⚡⚡⚡沸腾级 + 【喵娘锐评：钨价年内涨幅超60%，章源钨业6天4板领涨板块，量能持续放大】 + 【风险等级：🟡中风险】
**🔄 状态转换**：正从 **【供给过剩】** 向 **【供需紧平衡】** 迁移。

**🎯 驱动类型**：多重驱动
**🌍 国内外联合共振**：是（澳大利亚钨矿停产+国内收储预期+新能源需求爆发）
**🧩 关键拼图（变量）**：
*   [🟢S级] [2026-02-12]-自然资源部：2025年全国钨矿开采总量控制指标同比持平 （供给端政策管控力度不减） 🆕
*   [🟢S级] [2026-02-11]-澳大利亚钨矿：由于成本上升和环保压力，该公司宣布暂停钨矿开采业务 （全球钨矿供给收缩） 🆕
*   [🔵A级] [2026-02-12]-新华社：新能源汽车销量同比增长45%，动力电池需求旺盛 （下游需求持续增长） 📅
*   [🟡B级] [2026-02-12]-上海有色金属网：钨价报18.5万元/吨，年内涨幅超60% （价格持续上涨验证供需紧张） 🆕
*   **`🔍 质疑点`**：下游企业对高价钨的接受度如何？是否存在替代材料技术突破的可能性？收储政策落地的确定性和时间节点？
*   **`⚠️ 风险点`**：1) 收储政策可能不及预期；2) 钨价过快上涨可能抑制下游需求；3) 板块短期涨幅过大可能引发获利了结。
**🗺️ 作战地图**：
*   **🎯 核心突击位**：`002378 章源钨业` - 国内钨业龙头，资源自给率最高，**风险可控**。
*   **🧩 侧翼观测位**：`000657 中钨高新` - 钨粉加工龙头，**需进一步验证**。
*   **🧭 关键路标**：1) 钨价能否突破20万元/吨；2) 章源钨业能否突破前高8.5元；3) 板块涨停家数能否达到8家以上。
*   **🚨 风险路标**：1) 收储政策落空；2) 钨价出现回调；3) 大盘系统性风险。
**☯️ 喵娘洞见（六维推演）**：
1.  **🌱 生态位变化**：钨作为稀缺战略金属，供给收缩+需求爆发导致产业链价值重新分配，上游资源企业话语权显著增强。
2.  **🎭 主力剧本推演**：近期资金持续流入小金属板块，章源钨业作为龙头已走出趋势性行情，预计主力会围绕龙头反复炒作，补涨机会值得关注。
3.  **⏰ 节点必然性**：新能源汽车渗透率持续提升+硬质合金刀具需求复苏，三重因素叠加催生当前供需紧张格局，3-4月有望迎来传统旺季。
4.  **⚠️ 风险边界分析**：风险边界在钨价18万元/吨附近，若跌破则供需格局可能重新平衡；政策风险主要观察2月底两会是否有相关表述。
5.  **🌍 国内外联合共振分析**：澳大利亚钨矿停产（供给收缩）+国内收储预期（政策催化）+新能源需求爆发（需求增长），三重因素形成共振。
6.  **🚀 多重驱动分析**：政策驱动（收储）+ 国外驱动（澳矿停产）+ 技术驱动（新能源需求）+ 市场驱动（钨价上涨），驱动强度综合评分92分。
```
