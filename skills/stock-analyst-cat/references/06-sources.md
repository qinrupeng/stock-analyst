# 信源系统

本文档定义喵娘分析技能的信源等级标准和标注规范。

> **重要更新（2026-03-16）**：信源等级体系已升级为五级架构，新增S1/S2分级，明确评分权重。
>
> **重要更新（2026-03-28）**：新增"因果链标注规则"——S1/S2/A/B级是信源权威性等级，**必须叠加"因/果"标注**才能用于核心拼图。

---

## ⚠️ 因果链标注规则（2026-03-28新增）

> **核心原则**：S1/S2/A/B级是"信源权威性"等级，但用于关键拼图时，**必须区分"因"和"果"**

### "因"vs"果"定义

| 类型 | 定义 | 示例 | 核心拼图资格 |
|------|------|------|-------------|
| **因（驱动事实）** | 直接导致行情的原因 | 津巴布韦禁令升级、碳酸锂涨价、政府出台政策 | ✅ 可作为S/A级核心拼图 |
| **果（行情结果）** | 因导致的市场表现 | 融捷股份4连板、主力净流入98亿、板块涨停潮 | ❌ 不能作为S/A级核心拼图 |

> **重要说明**：信源等级（S1/S2/A/B）是**权威性维度**，因/果标注是**信息类型维度**，两者相互独立。例如："主力净流入98亿"若来源于S2级媒体，则标注为"🟢S2级（果）"；若来源于C级数据服务商，则标注为"🟠C级（果）"。

### 因果链判断规则

```
问：这个信息是"为什么涨"还是"涨了多少"？
- 回答"为什么涨" → 因（驱动事实）
- 回答"涨了多少" → 果（行情结果）
```

### 降级对照表

| 原错误标注 | 正确标注 | 原因 |
|------------|----------|------|
| 🟢🟢S1级 融捷股份4连板 | **S2级**（果） | 4连板是结果，不是驱动 |
| 🟢🟢S1级 主力净流入98亿 | **S2级**（果） | 资金流入是现象，不是原因 |
| 🟢🟢S1级 板块涨停潮 | **S2级**（果） | 涨停潮是板块表现，不是催化 |
| 🟢🟢S1级 黄金突破4550美元 | **S2级**（果） | 价格是结果，冲突升级才是因 |

### 因果链溯源检查清单（输出前必查）

```
□ 1. 这个信息是"为什么"还是"怎么样"？
□ 2. 如果没有这个事实，行情还会发生吗？（能→因，不能→果）
□ 3. 找得到这个信息的原始驱动吗？（找不到→降级为S2/A级）
□ 4. 信源是"谁说的"还是"发生了什么"？（混用→降级）
```

### 正确标注格式

```
# 因（驱动事实）—— 可作为S/A级核心拼图
[🟢🟢S1级] [2026-03-27]-津巴布韦官方：锂矿出口禁令升级至全面禁止 （直接驱动）

# 果（行情结果）—— 降级为S2/A级
[🟢S2级] [2026-03-27]-东方财富：融捷股份4连板 （趋势确认，非驱动）

# 注意：行情数据（涨停/资金/价格）一律降级处理
```

## 信源等级定义

| 等级 | 标识 | 定义 | 示例 | 评分权重 |
|------|------|------|------|----------|
| **S1级** | 🟢🟢 | 国家级官方机构 | .gov.cn、国务院、央行、证监会、发改委、工信部 | +15分 |
| **S2级** | 🟢 | 国家级权威媒体 | 求是杂志、人民日报、新华社、央视新闻、经济日报 | +10分 |
| **A级** | 🔵 | 专业财经媒体+部委官网 | 第一财经、财新、21世纪经济报道、证券时报、工信部官网 | +5分 |
| **B级** | 🟡 | 地方媒体/行业媒体 | 地方日报、行业网站、自媒体 | 不加分 |
| **C级** | 🟠 | 数据服务商（仅数据引用） | Wind、Choice、东方财富、同花顺 | 不加分 |

## 信源标注完整格式

```
[信源等级标识] [YYYY-MM-DD]-信源名称：[信息精要] （对市场生态的冲击解读） {时效标识}
```

### 格式说明

| 部分 | 说明 | 示例 |
|------|------|------|
| 信源等级标识 | 标准等级emoji | [🟢🟢S1级] / [🟢S2级] / [🔵A级] / [🟡B级] / [🟠C级] |
| 日期 | YYYY-MM-DD格式 | [2026-02-12] |
| 信源名称 | 发布机构或媒体 | -自然资源部 |
| 信息精要 | 核心信息提炼 | ：2025年全国钨矿开采总量控制指标同比持平 |
| 冲击解读 | 对市场的影响分析 | （供给端政策管控力度不减） |
| 时效标识 | 时间新鲜度 | 🆕 |

### 完整示例
```
[🟢🟢S1级] [2026-02-12]-自然资源部：2025年全国钨矿开采总量控制指标同比持平 （供给端政策管控力度不减） 🆕
[🟢S2级] [2026-03-16]-求是杂志：加快发展新质生产力 （国家战略级政策，科技板块重大利好） 🆕
```

## 时效性标注

| 标识 | 时间范围 | 使用场景 |
|------|----------|----------|
| 🆕 | 0-6小时 | 刚刚发布的重大消息 |
| 📅 | 6-24小时 | 今日消息，次新消息 |
| ⏰ | 24-72小时 | 隔日消息，仍有参考价值 |
| ❌ | >72小时 | 过时消息，参考价值低 |

## 信源排序规则

### 排序优先级
1. **等级排序**：S1级 → S2级 → A级 → B级 → C级（严格顺序）
2. **时间排序**：同等级内按时间倒序（最新优先）

### 正确示例
```markdown
*   [🟢🟢S1级] [2026-02-12]-自然资源部：... 🆕
*   [🟢S2级] [2026-03-16]-求是杂志：... 🆕
*   [🔵A级] [2026-02-12]-第一财经：... 📅
*   [🟡B级] [2026-02-12]-上海有色金属网：... 🆕
```

## 信源交叉验证规则（2026-03-26复盘新增）

> **复盘发现**：B级信源占比过高（40%），可能导致信息精度不足
> **案例**：百家号信源质量参差，需二次验证

### 交叉验证规则

| 信源等级 | 采用条件 | 验证要求 |
|----------|----------|----------|
| **S1/S2级** | 直接采用 | 无需验证 |
| **A级** | 与S级交叉验证后采用 | 需要S级确认 |
| **B级** | 需2个以上独立B级信源交叉验证 | 需要B级×2+ |
| **C级** | 不单独采用，需上级信源确认 | 需A/S级确认 |

### 交叉验证检查清单

```
□ 信源发布时间（是否最新）
□ 信源独立性（不同来源）
□ 内容逻辑一致性
□ 历史准确率记录
□ 是否有S/A级信源交叉确认
```

### B级信源升级规则

| 条件 | 升级后 | 说明 |
|------|--------|------|
| B级×2+ 且内容一致 | 视为A级 | 多个独立B级验证 |
| B级 + S/A级确认 | 视为A级 | 上级信源确认 |
| B级 + 市场走势验证 | 可用（降权） | 需要走势确认 |

### 禁止采用场景

```
❌ 单一B级信源 → 必须交叉验证
❌ B级信源内容矛盾 → 停止采用
❌ B级信源 + 与已知事实矛盾 → 停止采用
❌ C级信源单独使用 → 必须上级确认
```

## 冲击解读撰写规范

### 要求
- 必须说明信息对市场的具体影响
- 避免简单的信息重复
- 聚焦产业链、供需关系、估值影响等维度

### 示例
- ❌ 错误：澳大利亚钨矿宣布暂停开采
- ✅ 正确：澳大利亚钨矿宣布暂停开采（全球钨矿供给收缩，钨价有望获得支撑）
