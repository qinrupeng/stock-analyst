## ⚠️ 盯盘工作流（3步）

### 第1步：根据股票名称查secid（必须先执行）
```bash
curl -s "https://searchapi.eastmoney.com/api/suggest/get?input=股票名称&type=14&token=D43BF722C8E33BDC906FB84D85E326&count=5"
# 返回中取 QuoteID 字段，即为 secid
# 例如: "QuoteID":"0.300394" → secid=0.300394
```

### 第2步：获取实时数据（用第1步得到的secid）
```bash
# 2.1 实时行情
curl -s "https://push2.eastmoney.com/api/qt/stock/get?secid=替换这里&fields=f43,f44,f45,f46,f47,f48,f57,f58,f60,f116,f117,f169,f170,f171,f174,f176&ut=fa0e9c30&fltt=2&invt=2"

curl -s "https://push2.eastmoney.com/api/qt/stock/get?secid=替换这里&fields=f62,f78,f84,f164,f166,f168,f170,f172,f174,f176,f184&ut=fa0e9c30&fltt=2&invt=2"

# 2.3 MACD/RSI技术指标
curl -s "https://push2.eastmoney.com/api/qt/ulist.np/get?fltt=2&invt=2&fields=f12,f14,f23,f24,f25,f67,f68,f69,f100,f101,f103&secids=替换这里&ut=fa0e9c30"
```

## 四、热点生命周期判断标准

| 阶段 | 特征 | 信号 |
|------|------|------|
| 启动期 | 板块首次异动，消息刺激 | 板块涨幅>3%，个股涨停 |
| 爆发期 | 板块持续上涨，龙头连板 | 连续3日板块资金净流入 |
| 高潮期 | 板块全民讨论，媒体聚焦 | 日换手率>15%，散户大量进场 |
| 退潮期 | 龙头炸板，板块分化 | 主力净流出，个股开始跌停 |

| 指标 | 金叉信号 | 死叉信号 | 背离信号 |

| RSI | RSI<30后上穿50 | RSI>70后下穿50 | 价格新低但RSI未新低 |
| 量比 | 量比>1.5放量 | 量比<0.8缩量 | 价涨量缩/价跌量增 |
