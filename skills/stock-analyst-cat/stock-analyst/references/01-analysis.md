# 分析逻辑

## 分析维度详解

### 1. 消息面驱动分析

#### 政策驱动

**评估要点**：
- 政策级别（国务院/部委/地方）
- 政策力度（纲领性/操作性）
- 受益程度（直接/间接）
- 持续时间（短期/中期/长期）

**典型案例**：
- 新能源汽车补贴政策 → 整车/电池板块
- 半导体扶持政策 → 芯片/设备板块
- 房地产松绑政策 → 地产/建材板块

#### 技术驱动

**评估要点**：
- 技术成熟度（实验室/商用/成熟）
- 市场规模（百亿/千亿/万亿）
- 落地时间（近期/中期/远期）
- 竞争格局（蓝海/红海）

**典型案例**：
- AI大模型突破 → 算力/应用板块
- 固态电池量产 → 电池/材料板块
- 人形机器人商用 → 机器人/传感器板块

#### 业绩驱动

**评估要点**：
- 业绩确定性（已公告/预期/推测）
- 业绩弹性（微利/盈利/暴增）
- 持续性（一次性/可持续）
- 估值匹配（PE/PEG）

**典型案例**：
- 年报预增 → 当期业绩确认
- 订单落地 → 未来业绩保障
- 产品涨价 → 盈利能力提升

#### 事件驱动

**评估要点**：
- 事件影响力（行业/市场/全球）
- 预期差（市场是否充分预期）
- 持续性（一次性/反复炒作）
- 参与难度（适合机构/散户）

**典型案例**：
- 并购重组 → 乌鸡变凤凰
- 股权激励 → 业绩解锁条件
- 产品发布 → 新增长点

### 2. 市场预期分析

#### 资金流向

**关键指标**：
- 主力资金净流入（亿）
- 超大单净流入（亿）
- 大单净流入（亿）
- 散户净流出（亿）

**判断标准**：
- 净流入>5亿：强
- 净流入1-5亿：偏强
- 净流入-1~1亿：中性
- 净流入<-1亿：弱

#### 板块热度

**关键指标**：
- 板块涨幅（%）
- 板块换手率（%）
- 涨停家数
- 资金净流入（亿）

**判断标准**：
- 涨幅>5% + 换手率>15%：火热
- 涨幅3-5% + 换手率10-15%：较热
- 涨幅1-3% + 换手率5-10%：温和
- 涨幅<1% + 换手率<5%：清淡

#### 机构预期

**关键指标**：
- 研报数量
- 目标价涨幅
- 评级分布（买入/增持/中性）

**判断标准**：
- 买入评级>70%：机构看多
- 增持+买入>70%：机构偏多
- 中性>50%：机构中性
- 减持+卖出>30%：机构看空

### 3. 情绪驱动分析

#### 舆情热度

**关键指标**：
- 微博讨论量
- 股吧热度
- 搜索指数
- 新闻报道量

**判断标准**：
- 讨论量暴增：过热
- 讨论量较高：活跃
- 讨论量平稳：正常
- 讨论量低迷：冷清

#### 市场情绪周期

**阶段特征**：
- **复苏期**：分歧较大，机构建仓
- **狂热期**：一致性看多，场外资金涌入
- **分歧期**：涨势放缓，多空分歧
- **退潮期**：一致性看空，获利了结

## 分析框架

```
热点分析 = 消息面(45分) + 市场预期(25分) + 情绪驱动(30分)

消息面 = 政策(15) + 技术(10) + 业绩(10) + 事件(5) + 国外突破(5)
市场预期 = 资金(15) + 板块(10)
情绪驱动 = 舆情(15) + 周期(10) + 情绪传染(5)
```

> **风险分级**：风险分级（🟢低/🟡中/🟠高/🔴极高）独立评估，不计入热点评分

## 输出要求

每个热点分析必须包含：
1. **核心观点**：一句话概括投资逻辑
2. **驱动类型**：明确是哪种驱动
3. **态势评级**：核爆/沸腾/发酵/观望
4. **作战地图**：核心/侧翼/路标
5. **风险提示**：主要风险点
6. **操作建议**：仓位/时机/标的
