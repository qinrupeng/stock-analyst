# 集成测试执行指南

> 本文档指导如何执行stock-analyst-cat的集成测试

## 测试前准备

1. 确认三个Skill文件已创建：
   - `stock-editor/SKILL.md`
   - `stock-analyst/SKILL.md`
   - `stock-reviewer/SKILL.md`

2. 确认搜索工具可用：
   - mcporter 已配置
   - Browser 搜索可用

## 测试执行步骤

### Step 1: 加载主编Skill

```
用户输入："帮我做盘前扫描"
```

**执行动作**：
1. 加载 stock-editor/SKILL.md
2. 解析场景 → pre_market
3. 识别热点列表

### Step 2: 执行并行搜索

**执行动作**：
1. 使用mcporter并行搜索热点
2. 收集搜索结果
3. 执行轻审检查

**轻审检查项**：
- [ ] 有今日数据
- [ ] 热点数量≥3

### Step 3: 调用分析代理

**执行动作**：
1. 加载 stock-analyst/SKILL.md
2. 对每个热点进行深度分析
3. 生成态势评级和作战地图

### Step 4: 调用审查代理

**执行动作**：
1. 加载 stock-reviewer/SKILL.md
2. 对分析报告进行质量审查
3. 给出审查结论

### Step 5: 汇总输出

**执行动作**：
1. 主编汇总所有结果
2. 格式化输出报告
