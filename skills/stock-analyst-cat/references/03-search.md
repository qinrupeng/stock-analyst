# 搜索策略

本文档定义喵娘分析技能的信息搜索策略和执行流程。

---

## 七层搜索架构

喵娘采用**七层搜索架构**，从宏观到微观、从正向到反向，层层递进构建完整信息拼图。

```
第0层：市场全景扫描    → 快速定位潜在热点
    ↓
第1层：官方确认层     → 验证权威支撑
    ↓
第2层：专业解读层     → 获取深度逻辑
    ↓
第3层：市场反应层     → 评估情绪资金
    ↓
第4层：交叉验证层     → 寻找反面证据
    ↓
第5层：时间序列层     → 追踪历史轨迹
    ↓
第6层：风险排查层     → 识别潜在风险
```

---

## 各层详细说明

### 第0层：市场全景扫描
- **目标**：快速定位当日所有潜在热点
- **搜索范围**：主流财经媒体、交易所公告、监管部门通知
- **输出**：潜在热点清单（不评分）
- **关键词**："涨停"、"涨幅榜"、"热点"、"政策"、"突破"
- **执行时间**：盘前8:00-8:30

### 第1层：官方确认层
- **目标**：验证热点的官方/权威信息源
- **搜索范围**：
  - 国务院网站 (.gov.cn)
  - 证监会网站 (.csrc.gov.cn)
  - 发改委网站
  - 工信部网站
  - 交易所官网
  - 央行官网
- **目的**：确认热点是否具有权威支撑
- **信源等级**：S级

### 第2层：专业解读层
- **目标**：获取专业人士的深度解读
- **搜索范围**：
  - 券商研报
  - 专业财经媒体分析
  - 行业专家观点
- **目的**：理解热点的深层逻辑和影响
- **信源等级**：A级/B级

### 第3层：市场反应层
- **目标**：捕捉市场对热点的实时反应
- **搜索范围**：
  - 股票社区讨论
  - 投资者论坛
  - 社交媒体讨论
- **目的**：评估市场情绪和资金动向
- **关注点**：资金流向、板块涨跌、涨停数量

### 第4层：交叉验证层
- **目标**：寻找反面证据和质疑声音
- **搜索范围**：
  - 批评性报道
  - 质疑文章
  - 空方观点
- **目的**：验证热点的真实性和持续性
- **态度**：保持质疑精神，寻找反面证据

### 第5层：时间序列层
- **目标**：追踪热点的历史演变轨迹
- **搜索范围**：
  - 历史新闻
  - 过往研报
  - 早期公告
- **目的**：判断热点处于生命周期的哪个阶段
- **应用**：识别先导信号，判断持续性

### 第6层：风险排查层
- **目标**：识别潜在风险因素
- **搜索范围**：
  - 风险提示公告
  - 监管问询函
  - 负面新闻
- **目的**：全面评估风险收益比
- **输出**：风险信号清单

## 搜索执行策略

### 盘前搜索策略（8:00-9:00）

#### 步骤1：第0层市场全景扫描
1. 搜索关键词组合：
   - "今日涨停"、"涨幅榜"、"热点板块"
   - "政策发布"、"重大利好"
   - "技术突破"、"业绩超预期"
2. 输出：潜在热点清单

#### 步骤2：第1层官方确认
1. 对每个潜在热点，搜索官方信源
2. 确认政策/消息的权威性
3. 输出：经过官方验证的热点清单

#### 步骤3：第2层专业解读
1. 搜索券商研报深度解读
2. 搜索专业媒体分析
3. 输出：带有深度逻辑的热点清单

#### 步骤4：建立逻辑链条
1. 串联各层信息
2. 建立因果关系
3. 输出：完整逻辑链条

### 盘中搜索策略（9:30-14:00）

#### 持续监控
1. 跟踪热点演变
2. 捕捉新驱动因素
3. 实时风险排查

#### 动态调整
1. 根据市场变化调整评分
2. 更新态势评级
3. 修正作战地图

## 智能日期识别机制

### 日期识别逻辑
```python
def 获取搜索日期():
    today = datetime.now()
    weekday = today.weekday()
    
    if weekday == 0:  # 周一
        # 搜索上周五 + 周末两天
        return {
            "主时间窗口": (today - timedelta(days=3)).strftime("%Y-%m-%d"),  # 上周五
            "补充时间窗口": [
                (today - timedelta(days=2)).strftime("%Y-%m-%d"),  # 周六
                (today - timedelta(days=1)).strftime("%Y-%m-%d"),  # 周日
            ],
            "实时时间窗口": today.strftime("%Y-%m-%d")  # 今日
        }
    else:
        # 搜索昨天 + 今日
        return {
            "主时间窗口": (today - timedelta(days=1)).strftime("%Y-%m-%d"),  # 昨天
            "补充时间窗口": [],
            "实时时间窗口": today.strftime("%Y-%m-%d")  # 今日
        }
```

### 时间窗口应用
| 场景 | 主时间窗口 | 补充时间窗口 | 实时时间窗口 |
|------|------------|--------------|--------------|
| 周一盘前 | 上周五+周末 | - | 今日 |
| 平日盘前 | 昨日 | - | 今日 |
| 盘中 | 近日 | 近日+实时 | 实时 |
| 盘后 | 今日 | 近日 | - |
