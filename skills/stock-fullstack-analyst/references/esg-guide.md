## 📊 ESG评分方法

### E（环境）评分
```python
def evaluate_environmental(company_data: dict) -> dict:
    """
    环境(E)评分维度
    
    评分项目：
    1. 碳排放管理（0-25分）
       - 碳排放强度（单位营收碳排放）
       - 碳排放趋势（同比变化）
       - 碳中和承诺与计划
    
    2. 能源效率（0-25分）
       - 能耗强度
       - 可再生能源使用比例
       - 节能投资
    
    3. 废弃物管理（0-25分）
       - 废弃物排放达标率
       - 废弃物回收利用率
       - 污染防治投入
    
    4. 水资源管理（0-25分）
       - 用水强度
       - 水资源循环利用
       - 水风险应对措施
    
    数据来源：
    - 公司ESG报告
    - 环保部门披露
    - 第三方评级机构（MSCI、Sustainalytics）
    """
    return {
        "total_score": int,       # 总分(0-100)
        "carbon_emission": {
            "score": int,         # 得分(0-25)
            "intensity": float,   # 碳排放强度(tCO2e/百万营收)
            "trend": "up/down/stable",
            "carbon_neutral": bool
        },
        "energy_efficiency": {
            "score": int,
            "renewable_ratio": float,  # 可再生能源占比
            "energy_intensity": float
        },
        "waste_management": {
            "score": int,
            "recycling_rate": float,
            "compliance_rate": float
        },
        "water_resource": {
            "score": int,
            "water_intensity": float,
            "recycling_rate": float
        },
        "rating": "AAA/AA/A/BBB/BB/B/CCC",  # 评级
        "industry_rank": int                # 行业排名
    }
```

## 🏆 ESG评级标准

### 综合ESG评分
```python
def calculate_esg_rating(e_score: int, s_score: int, g_score: int) -> dict:
    """
    计算综合ESG评分
    
    权重分配（示例）：
    - E：30%
    - S：20%
    - G：50%（A股G权重通常较高）
    
    综合得分 = E×0.3 + S×0.2 + G×0.5
    
    评级划分：
    - AAA：90-100分（前10%）
    - AA：80-89分（前10-30%）
    - A：70-79分（前30-50%）
    - BBB：60-69分（前50-70%）
    - BB：50-59分（前70-90%）
    - B：40-49分（前90-97%）
    - CCC：<40分（后3%）
    """
```

## 🎯 ESG评分应用

### ESG增强选股模型
```
评分权重：
• 基本面评分：70%
• ESG评分：20%
• 技术面评分：10%

筛选标准：
• ESG评分 > 行业平均
• 无重大ESG负面事件
• E/S/G单维度无明显短板
```

### ESG风险过滤
```
排除标准：
• 环境：重大污染事件、环评违规
• 社会：重大安全事故、产品召回
• 治理：财务造假、高管违规

观察名单：
• ESG评分持续下滑
• 存在未决ESG争议
• ESG信息披露不完整
```
