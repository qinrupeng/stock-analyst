## 热点推荐机制

### 催化强度综合评估（2026-03-25复盘修正）

> **修正说明**：原方案"国际会议>国内政策"过于绝对，改为综合评估更合理

**禁止教条排序！催化强度必须综合评估以下维度：**

| 评估维度 | 权重 | 评估要点 |
|----------|------|----------|
| 时效性 | 30% | 当日催化>3日>7日>已消化 |
| 规模性 | 25% | 千亿市场>万亿市场 |
| 确定性 | 25% | 政策明确>预期朦胧 |
| 联动性 | 20% | 多板块共振>单一板块 |

**综合评估原则：**
- 国际会议催化不一定比国内政策催化强（国产替代政策可能更强）
- "首次提出"政策 > 已有政策延续
- 多板块共振 > 单一个股
- 业绩兑现 > 主题炒作

**催化强度评估公式：**
```
催化强度 = 时效性得分(30%) × 规模性得分(25%) × 确定性得分(25%) × 联动性得分(20%)
```

> **与02-scoring.md"催化时效性系数"的关系**：
> - 本节：综合催化强度（四维度加权），用于推荐优先级排序
> - 02-scoring.md：催化时效性系数（时间衰减系数），用于对"消息面驱动"维度评分进行时间衰减
> - 两者互补，不矛盾：先用本节评估综合强度，再用02-scoring的时效系数调整评分

### 风险过滤规则（必须先执行）

#### 禁止推荐规则（一票否决）【2026-03-23统一】
```
□ 板块风险等级≥🔴极高风险 → 禁止推荐
□ 板块连续3日涨幅>5% + 涨停>10只 → 禁止推荐
□ 板块换手率>15% → 禁止推荐
□ 资金流出信号明确 → 禁止推荐
□ 地缘风险爆发 → 禁止推荐
```

#### 限制推荐规则（条件触发）
```
□ 风险等级≥中风险 → 标注"⚠️ 风险提示"
□ 资金流入≤10亿 → 建议"轻仓参与"
□ 上涨天数≥5天 → 建议"谨慎追高"
```

### 推荐条件
1. **总分≥75分**：达到推荐门槛（根据风险等级动态调整阈值）
2. **驱动类型明确**：政策驱动/技术驱动/市场驱动/国外驱动/情绪驱动/多重驱动
3. **逻辑链条清晰**：因果关系明确
4. **风险可控**：风险等级≤🟡中风险（原≤🟠高风险）

### 推荐优先级
| 评级 | 优先级 | 行动建议 |
|------|--------|----------|
| 核爆级 | ⚡⚡⚡⚡⚡ | 重点关注，积极参与 |
| 沸腾级 | ⚡⚡⚡⚡ | 重点关注，适度参与 |
| 发酵级 | ⚡⚡⚡ | 关注，谨慎参与 |
| 观望级 | ⚡⚡ | 观察为主 |
| 高风险预警级 | ⚡ | 回避或极小仓位 |

## 热点白名单管理机制

### 入池标准
1. **总分≥75分**：达到推荐门槛
2. **多重驱动热点**：政策+技术+市场多重驱动优先
3. **国内外联合共振**：国外技术突破+国内跟进
4. **情绪到位+催化剂到位**：双重确认
5. **龙头明确**：有明确的板块龙头
6. **逻辑清晰**：逻辑链条完整，因果关系明确

### 出池标准
1. **热度下降**：涨停股减少，资金流出
2. **逻辑证伪**：关键逻辑被证伪
3. **风险暴露**：出现重大风险信号
4. **评级下调**：态势评级下调至观望级以下

### 动态调整
1. **每日更新**：每日根据最新信息更新白名单
2. **实时监控**：盘中持续监控热点变化
3. **及时调整**：根据市场变化及时调整推荐

## 利好阶段判断规则（2026-03-20复盘新增）

### 触发条件
当发现重大事件/政策/会议催化时，必须判断利好所处阶段

### 利好阶段判断标准

| 阶段 | 时间特征 | 操作建议 |
|------|----------|----------|
| **预期阶段** | 事件发生前3-7天 | 可提前布局，观察资金流向 |
| **发酵阶段** | 事件发生前1-3天 | 积极参与，趋势持有 |
| **兑现阶段** | 事件发生当天 | 警惕利好出尽，谨慎追高 |
| **出尽阶段** | 事件发生后1-3天 | 禁止追高，防止高位接盘 |

### 强制检查项
```
□ 利好事件距离当前时间？
□ 利好是否已被市场充分反映？
□ 资金是否在借利好出货？
□ 是否存在"利好出尽"风险？
□ 当前处于哪个阶段？
```

### 利好阶段识别量化标准

| 阶段 | 舆情热度 | 资金流向 | 涨幅特征 |
|------|----------|----------|----------|
| 预期阶段 | 日增<5篇 | 资金小幅流入 | 温和上涨<5% |
| 发酵阶段 | 日增5-20篇 | 资金持续流入 | 明显上涨5-15% |
| 兑现阶段 | 日增>20篇 | 资金开始流出 | 加速上涨>15% |
| 出尽阶段 | 热度骤降 | 大幅净流出 | 高位震荡/下跌 |

### 案例分析

**华为大会（2026-03-19-20）**：
- 事件时间：3月19-20日召开
- 盘前时间：3月20日早
- 所处阶段判断：**兑现阶段**（事件当天）
- 操作建议：警惕利好出尽，禁止作为核心突击位
- 实际情况：拓维信息跌幅6.71%，领跌中小100R指数

**华工科技公告（2026-03-18）**：
- 公告时间：3月18日晚发布"全球首发12.8T XPO光模块"
- 盘前时间：3月20日早
- 所处阶段判断：**发酵阶段**（公告后1-2天）
- 操作建议：应纳入热点池，积极参与
- 实际情况：华工科技涨4.54%，源杰科技20%涨停

### 与利好类型判断的关系
- **一次性利好**（政策出台、事件落地）→ 利好兑现=跑路
- **持续性利好**（涨价、春耕、业绩持续）→ 可以格局，持有为主

> **重要**：利好阶段判断是热点推荐的前置检查，必须在推荐前明确当前所处阶段

## 板块内部分化分析（2026-03-24复盘优化）【重要】

**触发条件：** 当热点板块涨幅较大时（涨停股≥5只或涨幅≥5%连续3天）

**⚠️ 与高位股分类的边界说明：**
```
【边界区分】
   - 高位股分类：解决"这一个股能不能追"的问题
   - 板块内部分化：解决"板块机会在哪里"的问题
   - 两者是互补关系，不是替代关系
   - 高位股分类后，还要判断板块所处阶段来确认操作时机
```

### 【强制分析步骤】

**Step 1：板块整体扫描**
```
- 板块涨停股数量？
- 板块整体涨幅？
- 主力资金净流入？
```

**Step 2：识别板块内部结构**
```
┌─────────────────────────────────────────────────────────────┐
│ 角色       │ 特征                      │ 操作建议          │
├─────────────────────────────────────────────────────────────┤
│ 龙头股    │ 涨幅最大 + 启动最早       │ 观望（位置高）   │
│ 跟涨股    │ 涨幅适中 + 启动晚1-2天    │ 可参与           │
│ 补涨股    │ 涨幅较小 + 启动晚3-5天    │ 重点关注         │
│ 滞涨股    │ 涨幅最小或下跌            │ 可能有雷         │
└─────────────────────────────────────────────────────────────┘
```

**Step 3：判断板块所处阶段**
```
┌─────────────────────────────────────────────────────────────┐
│ 阶段       │ 特征                           │ 操作建议      │
├─────────────────────────────────────────────────────────────┤
│ 启动期    │ 龙头刚启动，跟涨股未动         │ 布局龙头      │
│ 发酵期    │ 龙头涨，跟涨股开始跟涨         │ 布局跟涨      │
│ 高潮期    │ 龙头高位，补涨股全面爆发       │ 布局补涨，谨慎│
│ 退潮期    │ 龙头开始跌，滞涨股大跌         │ 远离          │
└─────────────────────────────────────────────────────────────┘
```

**Step 4：资金轮动检测**
```
- 资金是否从龙头流向补涨股？
- 是否有新资金入场？
- 量化资金是否在出货？
```

**【决策规则】**
```
- 启动期/发酵期：机会在龙头和跟涨股
- 高潮期：机会在补涨股，龙头需谨慎（参考高位股判断）
- 退潮期：远离，等待下一轮
```

### 【案例验证】

**深海科技案例（2026-03-24复盘数据）：**
| 检查项 | 实际情况 | 判断 |
|--------|----------|------|
| 板块涨停股数 | 海洋王涨停 | ✅ 板块强势 |
| 龙头（大连重工） | 涨+3.37% | ⚠️ 高位（需套用高位股分类） |
| 补涨股 | 海洋王涨停 | ✅ 补涨机会 |
| 所处阶段 | 高潮期 | ⚠️ 机会在补涨 |

**教训：** 深海科技高潮期，机会在补涨股（海洋王），大连重工属于高位震荡（需进一步套用高位股分类判断）

## 高位股分类标准（2026-03-24复盘优化）【重要】

**触发条件：** 当标的近期涨幅>40%或连续涨停≥3天

### 【强制分类检查】

**Step 1：识别高位特征】
```
- 近期涨幅是否>40%？
- 是否连续涨停≥3天？
- 达到任一条件即触发高位分析
```

**Step 2：分类检查（必须逐项检查）**
```
┌──────────────────────────────────────────────────────────────┐
│ 检查项          │ 数值/特征    │ 风险判断                  │
├──────────────────────────────────────────────────────────────┤
│ 换手率         │ >15% = 危险  │ 筹码松动，出货风险         │
│                │ 5-15% = 正常 │ 换手充分                  │
│                │ <5% = 稳定   │ 筹码锁定                  │
├──────────────────────────────────────────────────────────────┤
│ 主力资金       │ 净流出       │ 主力在跑，风险高           │
│                │ 净流入       │ 新资金接筹，风险中        │
│                │ 持平        │ 观望                      │
├──────────────────────────────────────────────────────────────┤
│ 龙虎榜         │ 机构卖出     │ 主力在出货                │
│                │ 游资卖出     │ 可能是对倒                │
│                │ 游资买入     │ 新资金进场                │
├──────────────────────────────────────────────────────────────┤
│ 封单金额       │ 递减         │ 资金在撤退                │
│                │ 递增         │ 封板坚定                  │
│                │ 不稳         │ 尾盘强行封板              │
└──────────────────────────────────────────────────────────────┘
```

**Step 3：高位股分类**
```
┌──────────────────────────────────────────────────────────────┐
│ 类型       │ 特征                           │ 操作建议        │
├──────────────────────────────────────────────────────────────┤
│ 出货型    │ 换手率>15% + 主力净流出        │ 禁止追高，立即减│
│           │ + 机构卖出 + 封单递减           │ 仓，远离       │
├──────────────────────────────────────────────────────────────┤
│ 换手型    │ 换手率5-15% + 主力净流入       │ 高位震荡，可观望│
│           │ + 新游资进场 + 封单稳定          │ 不建议追高     │
├──────────────────────────────────────────────────────────────┤
│ 锁仓型    │ 换手率<5% + 主力锁仓           │ 可持有，等机会 │
│           │ + 机构博弈 + 无龙虎榜            │ 不建议新进场   │
└──────────────────────────────────────────────────────────────┘
```

**Step 4：结合板块判断**
```
- 板块整体强势 → 高位股压力分散（可观望）
- 板块整体弱势 → 高位股孤掌难鸣（需谨慎）
- 板块内部轮动 → 机会在补涨股（参考板块分化分析）
```

### 【案例验证】
