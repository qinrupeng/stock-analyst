### 第1.3步：上市公司公告扫描（必调）→ 2026-03-20复盘新增

- 调用模块：`上市公司公告扫描机制`
- 强制动作：
  1. **扫描时间窗口**
     - 前一日晚间20:00-24:00发布的公告
     - 当日开盘前1小时发布的公告
  2. **扫描重点**
     - 新品发布、技术突破
     - 重大合同、中标
     - 业绩预告、高送转
     - 并购重组、股权变动
  3. **筛选标准**
     - 公告类型属于"重大利好"定义
     - 公告发布时间在扫描窗口内
  4. **输出成果**
     - 公告热点清单（股票代码+公告类型+利好强度）

- **重大利好量化定义**：
  ```
  重大利好 = 满足以下任一条件：
  □ 1. 新品/新技术发布，且明确提及业绩贡献预期
  □ 2. 重大合同签订，金额 ≥ 上年营收10%
  □ 3. 政策补贴/中标，金额 ≥ 上年净利润20%
  □ 4. 并购重组，涉及金额 ≥ 公司市值5%
  □ 5. 业绩预告，净利润同比增长 ≥ 50%
  ```

- **利好权重调整（2026-03-20复盘新增）**：
  ```
  公司个体重大利好权重 > 行业政策利好权重
  
  判断规则：
  - 有公司个体重大利好 + 前期涨幅>30% = 可纳入观察池，综合评估
  - 无公司个体重大利好 + 前期涨幅>30% = 直接排除
  ```

- **静默执行**：记录但不输出

### 第1.5步：盘前信号监控（必调）→ 2026-03-16复盘新增
- 调用模块：`盘前信号监控清单`
- 强制动作：
  1. **主线板块核心逻辑变化检测**
     - 搜索主线板块相关新闻
     - 判断逻辑是否被证伪/强化
  2. **低位板块利好汇总**
     - 搜索"政策+板块"关键词
     - 筛选实锤利好
  3. **ETF盘前溢价率监控**
     - 检查行业ETF盘前溢价
     - 溢价>0.5% = 资金看好信号
  4. **北向资金流向监控**
     - 检查北向资金行业流向
     - 净流入>5亿 = 外资看好信号
  5. **S级信源政策发布检查**
     - 检查求是/人民日报/新华社最新发布
     - 识别政策催化行业
  6. **核心标的流通市值统计**
     - 统计备选热点核心标的市值分布
     - 评估弹性空间
- 输出成果：盘前信号监控结果
- **静默执行**：记录但不输出

### 第2步：九层搜索架构完整执行（2026-03-21升级）→ 对应原始步骤2
- 调用模块：`九层搜索架构` 的 **0, 1, 2, 2.3, 2.5, 2.6, 2.7, 3, 4层**

  3. **必须执行情绪算法扫描（2.6层）**：调用NLP舆情分析，量化市场情绪，获取情绪得分
  4. **必须执行资金算法扫描（2.7层）**：调用L2数据追踪，识别主力资金行为，获取资金得分

### 第2.6步：跷跷板预判检测（必调）→ 2026-03-16复盘新增
- 调用模块：`跷跷板预判机制`
- 触发条件检测：
  □ 当前主线板块是否连续上涨≥3天，累计涨幅≥15%？
  □ 盘前该板块核心上涨逻辑是否被100%证伪（有官方消息或权威媒体实锤）？
  □ 是否有低位板块（过去2周累计回调≥8%）有实锤利好支撑？
  
- 如果满足全部条件 → 执行跷跷板预判流程：
  1. **筛选备选板块**：筛选所有低位、有实锤利好的板块，按「利好确定性 > 板块位置 > 情绪热度」排序，选出Top3进入备选池
  2. **评分调整**：给每个备选板块评分，情绪驱动项+10分，≥45分即可判定为「弱发酵级机会」，纳入观察名单
  3. **重点监控**：备选板块的ETF盘前溢价、北向资金净流入、主力资金挂单量
  
- 临盘确认（开盘15分钟内）：
  □ 原逻辑证伪板块主力净流出≥5亿，放量下跌？
  □ 备选板块主力净流入≥2亿，ETF盘前溢价≥0.5%？
  □ 同时满足 → 小仓位（≤5%）参与
  □ 不满足 → 放弃机会，不强行参与
  
- 输出成果：跷跷板预判结果（备选板块清单+评分+临盘确认条件）
- **静默执行**：记录但不输出
