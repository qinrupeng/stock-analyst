# 漏判防范专项

本文档定义漏判防范机制和检查清单。

---

## 漏判防范机制

### 漏判类型分析

| 漏判类型 | 特征 | 防范措施 |
|----------|------|---------|
| 周期性热点漏判 | 价格上涨、需求爆发被忽略 | 扫描成本推动、供需变化 |
| 事件驱动漏判 | 重大活动、政策发布被忽略 | 扫描关键时间节点 |
| 技术突破漏判 | 国外技术突破未识别 | 强制调用共振机制 |
| 情绪+催化剂漏判 | 情绪到位+催化剂触发被忽略 | 强制调用爆发识别 |

### 漏判防范工作流

#### 工作流A：周期性热点防范
```
触发：发现价格上涨/需求预期/库存变化信号
步骤：
1. 立即调用周期性热点扫描
2. 调用产业链逻辑扫描-成本推动部分
3. 调用变量定性-周期变量识别
4. 调用多重驱动分析-市场驱动部分
5. 输出周期性热点专项报告
```

#### 工作流B：事件驱动热点防范
```
触发：发现重大活动/政策发布/公司事件
步骤：
1. 立即调用事件驱动扫描
2. 调用变量定性-事件变量识别
3. 调用情绪+催化剂爆发式上涨识别机制
4. 调用态势评级-考虑事件影响
5. 输出事件驱动热点专项报告
```

#### 工作流C：技术突破热点防范
```
触发：发现国外技术突破/国内技术跟进
步骤：
1. 立即调用国内外联合共振识别机制
2. 调用多重驱动分析-技术驱动部分
3. 调用变量定性-技术突破变量识别
4. 调用态势评级-考虑技术突破影响
5. 输出技术突破热点专项报告
```

#### 工作流D：春耕周期热点防范（2026-03-06新增）
```
触发：每年3-4月
步骤：
1. 立即调用春耕周期扫描
2. 搜索磷肥/尿素价格数据
3. 识别供需缺口
4. 调用多重驱动分析-市场驱动部分
5. 输出周期性热点专项报告
```

#### 工作流E：政策解读快速响应（2026-03-06新增）
```
触发：重大政策发布（政府工作报告、两会期间）
步骤：
1. 立即搜索政策解读
2. 识别政策催化行业
3. 评估政策强度（一次性/持续性）
4. 输出政策解读报告
```

## 漏判防范检查清单

### 基础检查清单
```
□ 是否执行了产业链逻辑扫描（上游→中游→下游、横向关联、政策→产业链）？
□ 是否扫描了先行指标（订单/产能/价格）？
□ 是否验证了市场信号（异动/资金）？
□ 是否评估了舆情热度（媒体报道/社交媒体）？
□ 是否识别多重驱动（单一/双重/三重及以上）？
```

### 周期性热点专项检查
```
□ 是否扫描了价格上涨/成本推动关键词？
□ 是否扫描了供需变化/库存周期关键词？
□ 是否扫描了产能利用率/供应缺口关键词？
□ 是否识别了周期拐点信号？
```

### 事件驱动专项检查
```
□ 是否扫描了政策发布/新规出台关键词？
□ 是否扫描了重大活动/发布会关键词？
□ 是否扫描了公司公告/业绩预告关键词？
□ 是否评估了事件催化剂强度？
```

### 技术突破专项检查
```
□ 是否扫描了国外技术突破关键词？
□ 是否调用了国内外联合共振识别？
□ 是否扫描了国内跟进/产业共振？
□ 是否识别了增量市场空间？
```

### 情绪+催化剂专项检查
```
□ 是否评估了舆情热度（日增≥10篇）？
```

## 质疑清单（7项）

```
□ 我的判断是否有足够信源支撑？（S/A级信源≥2个）
□ 是否存在我忽略的风险信号？（搜索风险关键词）

□ 是否存在反面证据？（主动搜索负面信息）
□ 我的推演是否存在过度乐观？（考虑最坏情况）
```

## 漏判案例库

### 案例1：人形机器人漏判
- **漏判原因**：未识别春晚利好出台=庄家出货信号
- **教训**：一次性利好出台就是出货信号
- **防范措施**：增加利好类型判断

### 案例2：稀土漏判
- **漏判原因**：未区分一次性利好vs持续利好
- **教训**：利好一次性=高开低走
- **防范措施**：增加利好类型标注

### 案例3：磷化工正确案例
- **正确原因**：识别了多重持续催化（春耕+新能源）
- **经验**：持续催化>一次性利好

### 案例4：油气板块 - 历史经验陷阱（2026-03-03复盘新增）
- **错误类型**：确认偏差 + 思维定式
- **错误过程**：看到地缘冲突->触发利好出尽模式，看到三桶油涨停->判断利好出尽，忽视期货持续涨停、首次2连板、资金持续流入
- **实际走势**：三桶油历史首次2连板，油气板块涨停20+只，期货市场持续涨停
- **核心教训**：2026年地缘冲突新常态：不能简单用历史经验判断；历史首次信号出现时，应上调预期；期货市场是领先指标，需重视

### 案例5：光伏板块完全遗漏（2026-03-03复盘新增）
- **漏判原因**：未在盘前扫描中覆盖光伏板块，未搜索新能源政策相关关键词
- **实际情况**：盘后涨幅居前：光伏设备板块上涨3.28%，涨停股：欧晶科技涨停，催化剂：工信部等六部门联合发布《关于促进光伏组件综合利用的指导意见》
- **防范措施**：光伏板块每日必扫

### 案例6：四川黄金3连板后跌停（2026-03-07复盘新增）
- **漏判原因**：未识别"3连板=诱多信号"，未识别游资"拉高出货"模式
- **实际情况**：四川黄金3连板吸引散户追高，第4天直接跌停收割
- **防范措施**：3连板=诱多信号（危险），4连板及以上=必然出货（远离）

### 案例7：涨幅>75%红线（2026-03-07复盘新增）
- **漏判原因**：未识别涨幅过大风险
- **实际情况**：汉缆股份11天7板，涨幅>75% → 风险累积；聚飞光电2月至今涨幅>76% → 见顶信号
- **防范措施**：涨幅>75%不管逻辑多好，必跌

### 案例8：煤炭采掘板块完全遗漏（2026-03-03复盘新增）
- **漏判原因**：未关注到煤炭属于采掘行业，未搜索煤炭/能源股资金流向
- **实际情况**：煤炭开采行业整体上涨2.42%，主力资金净流入6.93亿元，兖矿能源涨8.75%
- **防范措施**：煤炭板块加入必扫清单

### 案例9：智能电网板块崩盘（2026-03-13复盘新增）
- **漏判原因**：
  1. 未识别"涨幅>60%预警"
  2. 未识别"板块连续大涨预警"
  3. 未识别"主力资金轮动信号"
  4. 线性外推"强者恒强"
- **实际情况**：
  - 顺钠股份：跌停（-9.98%）
  - 汉缆股份：大跌9.44%，龙虎榜显示机构净卖出1.27亿元
  - 电网设备概念集体下挫
- **漏判过程**：
  - 3月3日至12日：顺钠股份从11.18元涨到18.84元，累计涨幅68%
  - 盘前虽提示"短期涨幅35%较大，需注意回调"
  - 但仍将其列入推荐股票池，且放在了第一位
  - 未触发75%红线（68%<75%），但忽略了60%预警线
- **防范措施**：
  - 增加"涨幅60%预警线"
  - 增加"板块连续3日大涨预警"
  - 增加"主力资金轮动监控"
  - 强化资金行为分析权重

## 漏判防范专项规则（复盘整合）

### 规则1：地缘冲突新常态判断规则（2026-03-03复盘整合）

【新增判断标准】
□ 地缘冲突是否仍在持续发酵？
  - 持续发酵中 = 持续利好（可追高）
  - 已达高峰 = 一次性利好（高开低走）
  - 出现缓和迹象 = 混合（观察）

□ 期货市场信号（关键！）
  - 期货持续涨停 = 行情未结束
  - 期货高开低走 = 利好出尽

□ 历史首次信号
  - 首次出现 = 行情强度超预期
  - 谨慎使用历史经验判断

□ 资金行为验证
  - 主力净流出但板块涨停 = 分歧加大，可能继续涨
  - 主力净流入+板块涨停 = 资金推动，可能持续

【触发条件】
当发现以下信号时，必须触发地缘冲突新常态判断：
1. 出现历史首次（如三桶油首次2连板）
2. 期货市场持续涨停（连续2天以上）
3. 地缘冲突仍在持续，未见缓和
4. 板块涨停数量>10只

【强制检查项】
□ 是否出现历史首次信号？-> 是=主动上调评级
□ 期货市场是否持续涨停？-> 是=行情未结束
□ 地缘冲突是否仍在发酵？-> 是=不是一次性利好
□ 主力资金是否在买入？-> 是=持续催化

### 规则4：涨幅>75%红线规则（2026-03-07复盘整合）

  - 涨幅>75%+主力流入 = 泡沫风险
  - 涨幅>75%+高位放量 = 必跌信号

□ 是否有支撑涨幅的基本面？
  - 业绩增长>50% = 可支撑
  - 业绩增长<30% = 纯炒作

□ 是否有增量资金？
  - 主力持续流入>5天 = 还在涨
  - 主力开始流出 = 立刻跑

【强制检查项】
□ 涨幅是否>75%？
□ 基本面能否支撑涨幅？
□ 主力资金是否持续流入？

### 规则5：三重驱动筛选规则（2026-03-07复盘整合）

【新增判断标准】
□ 是否有三重驱动？
  - 政策+国外+业绩 = 三重（通过）
  - 政策+国外 = 两重（不通过）
  - 地缘+期货 = 两重（不通过）

□ 驱动强度是否递减？
  - 三重驱动持续增强 = 穿越
  - 两重驱动 = 一日游

□ 是否有持续催化？
  - 每日都有新催化 = 穿越
  - 催化一次性 = 结束

【强制检查项】
□ 是否为三重驱动？
□ 驱动是否持续增强？
□ 是否有每日新催化？

### 规则7：涨幅风险预警规则（2026-03-13复盘新增）

【新增判断标准】
□ 涨幅是否>60%？
  - 涨幅>60% = 黄色预警
  - 涨幅>60%+主力资金开始流出 = 立即减仓信号
  - 涨幅>60%+板块连续3日大涨 = 回调预警

□ 板块是否连续3日大涨？
  - 连续3日涨幅>5% = 过热预警
  - 涨停股数>10只 = 过热预警
  - 换手率>15% = 过热预警

□ 主力资金是否开始轮动？
  - 主力资金净流入连续3日后开始流出 = 出货信号
  - 主力资金大幅净流出（>100亿）= 系统性风险

【强制检查项】
□ 涨幅是否>60%？-> 是=从推荐池移除
□ 板块是否连续3日大涨？-> 是=发布过热预警
□ 主力资金是否开始流出？-> 是=触发减仓预警

### 规则8：跷跷板场景漏判防范（2026-03-16复盘新增）

【触发条件】
当发现以下信号时，必须触发跷跷板场景检测：
1. 当前主线板块连续上涨≥3天，累计涨幅≥15%
2. 盘前该板块核心上涨逻辑被100%证伪（有官方消息或权威媒体实锤）
3. 存在低位板块（过去2周累计回调≥8%）有实锤利好

【强制检查项】
□ 是否识别了主线逻辑证伪信号？
□ 是否筛选了低位、有实锤利好的备选板块？
□ 是否对备选板块进行了评分（情绪驱动项+10分）？
□ 是否设置了临盘确认条件（资金流出+流入信号）？

【漏判风险】
- 未提前将备选板块纳入观察池 → 错过资金跷跷板效应
- 未设置临盘确认条件 → 盲目参与，风险高
- 评分阈值过高 → 错过弱发酵级机会

### 规则9：小盘股流动性风险防范（2026-03-16复盘新增）

【触发条件】
当推荐标的流通市值<50亿时，必须执行流动性风险检查

【强制检查项】
□ 流通市值是否<30亿？-> 是=风险等级+1级，仓位上限≤10%
□ 流通市值是否<20亿？-> 是=需额外验证非庄股
□ 换手率是否>20%？-> 是=筹码松动风险
□ 是否连续涨停≥3天？-> 是=诱多风险，禁止推荐
□ 主力资金是否净流出？-> 是=出货风险，禁止推荐

【禁止推荐场景】
| 条件 | 原因 |
|------|------|
| 流通市值<20亿 + 换手率>20% | 庄股风险 |
| 流通市值<30亿 + 连续涨停≥3天 | 诱多风险 |
| 流通市值<30亿 + 主力资金净流出 | 出货风险 |

【漏判风险】
- 未识别小盘股流动性风险 → 止损难度大，亏损放大
- 未配套风险对冲规则 → 推荐高风险标的，用户亏损

### 案例12：华工科技公告遗漏（2026-03-20复盘新增）

- **漏判原因**：
  1. 盘前扫描未覆盖上市公司公告栏
  2. 仅关注行业政策，未关注公司个体重大利好
  3. 错误认为"前期涨幅高=风险大"，未考虑利好对估值的重塑

- **实际情况**：
  - 3月18日晚华工科技发布"全球首发12.8T XPO光模块"公告
  - 3月19日华工科技逆势上涨4.54%，主力净流入5.502亿
  - 盘前将其排除，错失机会

- **防范措施**：
  - 新增公告扫描环节（第1.3步）
  - 调整利好权重：公司个体重大利好 > 行业政策利好
  - 新增"重大利好"量化定义

### 规则11：公告扫描漏判防范（2026-03-20复盘新增）

【触发条件】
每次盘前扫描必须执行公告扫描环节

【强制检查项】
□ 是否扫描了巨潮资讯/交易所公告？
□ 是否识别了新品发布/重大合同类公告？
□ 是否将公告热点纳入热点池？
□ 是否调整了"前期涨幅高"标的的评级？

【漏判风险】
- 未扫描公告 → 遗漏公司个体重大利好
- 未调整利好权重 → 错误排除有重大利好的高位股

【落地验证机制】
每次复盘必须输出：
□ 1. 本次盘前扫描发现的公告数量
□ 2. 公告转化为热点的数量
□ 3. 公告遗漏导致的预测错误数量
□ 4. 优化前后准确率对比

连续3次未输出验证数据 → 触发流程审查

### 规则13：情绪异常识别规则（2026-03-23新增）

【触发条件】
当以下任一信号出现时，必须触发情绪异常检测：
1. 情绪得分日增≥20分（情绪突然升温）
2. 情绪得分日降≥20分（情绪突然降温）
3. 情绪极性极度分化（正面>70% 或 负面>70%）
4. 共识形成速度指数>80%（共识形成过快）
5. 情绪传染速度>100%（恐慌/亢奋快速扩散）

【强制检查项】
□ 情绪得分是否出现异常波动？（日增/日降≥20分）
□ 情绪极性是否极度分化？（正面/负面>70%）
□ 共识是否形成过快？（共识形成速度指数>80%）
□ 是否存在情绪传染现象？（情绪传染速度>100%）
□ 是否识别到量化资金介入？

【应对措施】
- 情绪突然升温：警惕诱多，降低仓位
- 情绪突然降温：警惕情绪退潮，准备止损
- 情绪极性极度分化：关注转折点，谨慎操作
- 共识形成过快：警惕反向风险，做好止盈
- 情绪传染扩散：立即评估风险等级，必要时清仓

【漏判风险】
- 未识别情绪异常 → 错过情绪转折点
- 未评估情绪传染 → 情绪扩散时反应滞后
- 未降低仓位 → 情绪退潮时亏损放大

### 规则14：量化踩踏风险预警（2026-03-23新增）

【触发条件】
当以下任一信号出现时，必须触发量化踩踏预警：
1. 板块内≥5只股票同时出现大单抛售（单笔>1亿）
2. 板块资金净流出≥50亿（单日）
3. 板块涨幅>60%后，换手率>20%（过热信号）
4. 多只涨停股同时出现"天地板"（涨停→跌停）
5. 量化资金行为识别显示"集体出逃"

【强制检查项】
□ 是否出现大单集体抛售信号？（≥5只股票，单笔>1亿）
□ 板块资金是否大幅净流出？（≥50亿）
□ 板块是否过热？（涨幅>60%+换手率>20%）
□ 是否出现"天地板"现象？（涨停→跌停）
□ 量化资金是否集体出逃？

【应对措施】
- 立即下调风险等级：高风险→极高风险
- 立即降低仓位：减仓50%或清仓
- 发布风险预警：明确提示量化踩踏风险
- 跟踪后续变化：监控是否出现连锁反应

【漏判风险】
- 未识别量化踩踏 → 亏损幅度放大
- 未及时减仓 → 错止损盈时机
- 未发布预警 → 用户反应滞后
