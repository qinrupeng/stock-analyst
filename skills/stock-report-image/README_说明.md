# 🎨 stock-report-image 技能使用说明
## 🌟 技能简介
这是**专为A股热点报告、今日头条股票文章、小红书财经内容定制的自动配图生成器**，基于baoyu-skills官方版本深度优化适配OpenClaw，比通用版更贴合财经场景，全程自动化无需手动写提示词。

### ✨ 核心优势
✅ 自动读取最新盘前/盘中报告，无需手动复制内容  
✅ 内置A股专属主题→风格映射，生成的图100%贴合财经场景  
✅ 自动适配头条/小红书/公众号多平台尺寸，不用手动调比例  
✅ 支持20种信息图布局、17种官方风格，可自由定制  
✅ 自动检测可用生图模型（imagen/autoglm），无需额外配置  
✅ 容错能力强，主题提取失败可手动补充，生图失败自动保存提示词

## 🚀 快速使用（两种方式）
### 方式1：自然语言触发（推荐，不用记命令）
直接说需求即可，自动触发技能：
```
// 基础用法
"给今天的盘前报告配个头条封面"
"给这篇稀土主题的文章配图"
"生成昨天热点报告的封面图"

// 平台适配
"生成小红书版的算力热点封面"
"做一张公众号用的行业分析封面"

// 定制需求
"用时间线布局生成新能源行业分析封面"
"用可爱风格生成AI主题的小红书配图"
```

### 方式2：命令行调用（手动跑脚本）
```bash
# 自动模式：读取最新报告→生成头条封面（最常用）
python scripts/generate_cover.py --auto

# 指定报告文件
python scripts/generate_cover.py --auto --report 2026-03-19-盘前扫描.md

# 仅生成提示词，不调用生图
python scripts/generate_cover.py --auto --prompt-only

# 生成小红书3:4竖版封面
python scripts/generate_cover.py --auto --platform xhs

# 强制指定风格+布局
python scripts/generate_cover.py --auto --style playful --layout timeline_horizontal
```

## ⚙️ 进阶参数说明
所有参数都是**可选**，不指定则自动适配最优值：

| 参数 | 作用 | 可选值/示例 | 默认值 |
|------|------|------------|--------|
| `--model` | 指定生图模型 | `imagen`（火山引擎）/ `autoglm`（智谱） | 自动检测可用模型，优先用imagen |
| `--platform` | 适配平台，自动调整尺寸比例 | `toutiao`（16:9横版）/ `xhs`（3:4竖版）/ `wechat`（2.35:1宽屏） | `toutiao`（头条16:9） |
| `--style` | 强制指定生成风格，跳过自动匹配 | 👉 完整17种风格见附录1 | 自动根据主题关键词匹配 |
| `--layout` | 指定信息图布局结构 | 👉 完整20种布局见附录2 | 自动根据主题数量适配 |
| `--prompt-only` | 仅生成提示词，不调用生图API | - | 关闭 |

## 📝 配置文件说明
所有可配置项都在 `config/styles_config.yaml` 中，无需改代码即可自定义：
### 1. 修改/新增主题→风格映射
```yaml
# 编辑 config/styles_config.yaml 中的 theme_styles 字段
theme_styles:
  # 新增主题直接在这里加
  AI代理: "blueprint_lab"  # 新增AI代理主题→科技蓝图风
  人形机器人: "acid_block" # 新增人形机器人主题→赛博朋克风
  算力: "blueprint_lab"    # 修改已有主题的风格
```
### 2. 调整默认参数
```yaml
default_style: "blueprint_lab"  # 修改默认风格
platform_aspect_ratios:
  xhs: "4:5"  # 自定义小红书尺寸比例
```

## ❓ 常见问题
### Q1：生图失败怎么办？
A：失败时会自动把提示词保存到输出目录的`.prompt.txt`文件里，可以手动复制提示词到其他生图工具生成，不会白跑。
### Q2：自动识别的主题不全/不对怎么办？
A：运行脚本时会自动提示是否补充主题，直接输入要补充的主题即可，也可以手动编辑配置文件加新的主题关键词。
### Q3：想加新的风格怎么办？
A：把新的风格参考图放到 `cover-image-styles/` 目录，然后在配置文件的 `style_ref_map` 里加映射即可。
### Q4：生成的图尺寸不对怎么办？
A：指定`--platform`参数即可，比如小红书用`--platform xhs`自动改成3:4竖版。
