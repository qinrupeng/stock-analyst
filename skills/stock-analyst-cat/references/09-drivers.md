# 热点驱动分析机制

本文档定义热点驱动类型的识别和分析机制。

---

## 情绪+催化剂爆发式上涨识别机制

### 识别条件
**情绪到位 + 催化剂到位 = 爆发式上涨**

### 情绪到位量化标准

| 评估维度 | 分值 | 阈值 | 说明 |
|----------|------|------|------|
| 市场关注度 | 10分 | 日增≥10篇 | 媒体报道数量 |
| 资金情绪 | 10分 | 连续3日流入 | 板块资金流向 |
| 板块热度 | 10分 | ≥5只 | 涨停股票数量 |
| 市场预期 | 10分 | 日增≥5篇 | 机构研报数量 |
| 社交媒体热度 | 5分 | 日增≥1000条 | 社交媒体讨论 |
| 投资者讨论 | 5分 | 热度提升 | 论坛、股吧讨论 |

**情绪得分≥30分 = 情绪到位**

### 催化剂到位量化标准

| 催化剂类型 | 分值 | 触发条件 |
|------------|------|----------|
| 国外技术突破 | 25分 | 马斯克/OpenAI等发布重大技术进展 |
| 重大政策发布 | 25分 | 证监会/发改委/工信部等发布重大政策 |
| 重大技术突破 | 25分 | 新技术研发成功、商业化应用突破 |
| 重大事件 | 25分 | 重大业绩、超预期订单等 |

**催化剂得分≥25分 = 催化剂到位**

### 爆发式上涨判定
- 情绪到位(≥30分) + 催化剂到位(≥25分) = 爆发式上涨
- 预期收益：短期涨幅≥20%
- 持续时间：3-5个交易日

## 国内外联合共振识别机制

### 共振定义
**国外技术突破 + 国内跟进 = 国内外联合共振**

### 共振条件量化

| 维度 | 分值 | 评估要点 |
|------|------|----------|
| 国外驱动 | 40分 | 有明确的国外技术突破 |
| 国内政策跟进 | 30分 | 有国内政策或市场跟进反应 |
| 产业链联动 | 30分 | 国内外产业链形成联动 |

**共振得分≥70分 = 国内外联合共振**

### 共振类型分类

#### 技术共振型
```
国外技术突破 → 国内技术跟进 → 产业链爆发
```
示例：OpenAI发布GPT-5 → 国内AI大模型跟进 → 算力/AI应用爆发

#### 政策共振型
```
国外政策变化 → 国内政策调整 → 市场反应
```
示例：美国芯片法案 → 国内半导体政策扶持 → 芯片板块爆发

#### 市场共振型
```
国外市场变化 → 国内市场联动 → 情绪传导
```
示例：特斯拉股价大涨 → 国内新能源车板块联动 → 情绪传导

## 多重驱动分析机制

### 多重驱动定义
**政策、技术、市场、国外、情绪中至少2个驱动类型 = 多重驱动**

### 多重驱动评分矩阵

| 驱动类型 | 分值 | 评估维度 |
|----------|------|----------|
| 政策驱动 | 20分 | 政策级别、落地确定性、资金支持力度 |
| 技术驱动 | 20分 | 技术领先性、商业化前景、产业链影响 |
| 市场驱动 | 20分 | 供需变化、价格波动、成交量变化 |
| 国外驱动 | 20分 | 国际市场变化、技术突破传导 |
| 情绪驱动 | 20分 | 市场情绪热度、投资者预期 |

### 多重驱动强度判定

| 强度等级 | 分数范围 | 特征描述 |
|----------|----------|----------|
| 史诗级多重驱动 | ≥90分 | 所有驱动同时发力，全面共振 |
| 强多重驱动 | ≥70分 | 4个以上驱动发力，形成合力 |
| 中等多重驱动 | ≥50分 | 3个驱动发力，形成合力 |
| 弱多重驱动 | ≥30分 | 2个驱动发力，局部共振 |

### 多重驱动分析示例

#### 示例1：电网设备板块
```
政策驱动(20分)：国家电网十五五4万亿投资
技术驱动(15分)：AI算力建设爆发
市场驱动(18分)：变压器订单排至2027年
情绪驱动(12分)：资金持续流入
合计：65分 → 强多重驱动
```

#### 示例2：AI算力板块
```
国外驱动(25分)：OpenAI发布GPT-5
政策驱动(20分)：国内AI政策扶持
技术驱动(20分)：算力需求爆发
情绪驱动(15分)：市场热度高涨
合计：80分 → 史诗级多重驱动
```
