# 热点发现与推荐机制

本文档定义喵娘分析技能的热点发现逻辑和推荐标准。

---

## 热点发现流程

### 步骤1：消息面扫描
1. 捕捉关键词（政策、业绩、事件、周期、情绪）
2. 识别信息类型
3. 初步判断热点潜力

### 步骤2：板块分类判断
根据关键词匹配板块类型：
- 政策驱动型
- 业绩驱动型
- 事件驱动型
- 周期型
- 情绪驱动型

### 步骤3：选择验证策略
调用对应板块的市场预期验证清单

### 步骤4：执行验证流程
按照验证清单逐项检查

### 步骤5：综合评分
根据验证结果给出热点评分

---

## 热点推荐机制

### 推荐条件
1. **总分≥70分**：达到推荐门槛
2. **驱动类型明确**：政策驱动/技术驱动/市场驱动/国外驱动/情绪驱动/多重驱动
3. **逻辑链条清晰**：因果关系明确
4. **风险可控**：风险等级≤🟠高风险

### 推荐优先级
| 评级 | 优先级 | 行动建议 |
|------|--------|----------|
| 核爆级 | ⭐⭐⭐⭐⭐ | 重点关注，积极参与 |
| 沸腾级 | ⭐⭐⭐⭐ | 重点关注，适度参与 |
| 发酵级 | ⭐⭐⭐ | 关注，谨慎参与 |
| 观望级 | ⭐⭐ | 观察为主 |
| 高风险预警级 | ⭐ | 回避或极小仓位 |

---

## 热点白名单管理机制

### 入池标准
1. **多重驱动热点**：政策+技术+市场多重驱动
2. **国内外联合共振**：国外技术突破+国内跟进
3. **情绪到位+催化剂到位**：双重确认
4. **龙头明确**：有明确的板块龙头
5. **逻辑清晰**：逻辑链条完整，因果关系明确

### 出池标准
1. **热度下降**：涨停股减少，资金流出
2. **逻辑证伪**：关键逻辑被证伪
3. **风险暴露**：出现重大风险信号
4. **评级下调**：态势评级下调至观望级以下

### 动态调整
1. **每日更新**：每日根据最新信息更新白名单
2. **实时监控**：盘中持续监控热点变化
3. **及时调整**：根据市场变化及时调整推荐

## 板块分类与智能匹配机制

### 板块类型识别流程
```
步骤1：消息面扫描
  → 识别关键词（政策、业绩、事件、周期、情绪）
  
步骤2：板块分类判断
  → 根据关键词匹配板块类型（政策驱动型/业绩驱动型/事件驱动型/周期型/情绪驱动型）
  
步骤3：选择验证策略
  → 调用对应板块的市场预期验证清单
  
步骤4：执行验证流程
  → 按照验证清单逐项检查
  
步骤5：综合评分
  → 根据验证结果给出市场预期得分
```

### 关键词匹配规则

#### 政策驱动型关键词
"政策发布"、"投资计划"、"国家支持"、"行业规划"、"财政补贴"、"国家电网"、"特高压"、"智能电网"

#### 业绩驱动型关键词
"业绩预告"、"超预期"、"净利润增长"、"营收增长"、"机构调研"

#### 事件驱动型关键词
"突发事件"、"疫情"、"灾害"、"重大事故"、"国际冲突"

#### 周期型关键词
"价格上涨"、"库存下降"、"开工率"、"供需关系"、"大宗商品"

#### 情绪驱动型关键词
"涨停潮"、"次新股"、"高送转"、"重组"、"概念炒作"
