## 二、A股赛道推荐映射

| 赛道主题 | 推荐风格 | 推荐布局 | 理由 |
|---------|---------|---------|------|
| 投资机会、赛道机会、热点前瞻 | `folder` ⭐ | `top3_bottom2` | 专业金融报告感，大爷验证最爱 |
| AI数字经济、芯片、算力 | `blueprint` / `intuition-machine` | `grid_2x3` | 科技蓝图/硬核科技感 |
| 有色金属、黄金、铜、稀土 | `vintage` / `retro` | `hub_spoke` | 复古贵金属感 |
| 储能锂电、光伏、电力 | `blueprint` | `grid_2x3` | 科技蓝图能量感 |
| 量子通信、航天、机器人 | `intuition-machine` / `dark_atmospheric` | `hub_spoke` | 前沿科技感 |
| 深海科技 | `folder` | `top3_bottom2` | 专业感 |
| 医药、医疗 | `elegant` / `watercolor` | `grid_2x3` | 专业温和 |
| 银行、金融 | `elegant` | `grid_2x3` | 权威古典感 |
| 化工、化肥 | `vintage` / `warm` | `grid_2x3` | 传统行业感 |

---

## 三、使用方法

1. 根据赛道主题，在上方表格找推荐风格key
2. 在 `styles_config.yaml` 的 `theme_styles` 中确认映射
3. 生成时指定风格：`python generate_cover.py 报告.md --style folder`
4. 指定布局：`python generate_cover.py 报告.md --layout top3_bottom2`
