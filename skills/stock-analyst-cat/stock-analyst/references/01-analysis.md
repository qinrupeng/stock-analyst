## 分析维度（引用02-scoring.md第三层）

### 四维评分体系

| 维度 | 权重 | 说明 | 评分依据 |
|------|------|------|---------|
| **情绪到位** | 30% | 市场情绪共鸣程度 | 媒体热度、讨论量、搜索指数 |
| **催化剂到位** | 25% | 利好兑现进度 | 政策落地、技术突破、业绩确认 |
| **逻辑强度** | 25% | 因果链完整性 | 政策力度、技术成熟度、业绩确定性 |
| **机构共识** | 20% | 机构认可程度 | 研报数量、评级分布、资金流向 |

### 驱动类型（引用02-scoring.md第一层）

| 驱动类型 | 热点类型 | 使用模式 |
|----------|----------|----------|
| **政策驱动型** | 政策出台、行业规划 | 理性模式 |
| **技术突破型** | 技术创新、商业化突破 | 混合模式 |
| **情绪驱动型** | 市场情绪主导 | 量化模式 |
| **资金驱动型** | 主力资金推动 | 量化模式 |
| **概念炒作型** | 主题投机、散户跟风 | 量化模式 |
| **国际事件型** | 国际事件驱动 | 混合模式 |

## 分析流程

### 步骤1：识别热点类型

根据信源特征和关键词判断：
- S1级政策/国务院 → 政策驱动型
- 技术突破/商用/量产 → 技术突破型
- 概念/炒作/龙头 → 情绪驱动型
- 主力流入/机构买入 → 资金驱动型
- 美股映射/国际会议 → 国际事件型

### 步骤2：确定评分模式

| 热点类型 | 推荐模式 | 权重配置 |
|----------|----------|----------|
| 政策驱动型 | 理性模式 | 消息面45%+市场预期25%+情绪驱动30% |
| 技术突破型 | 混合模式 | 智能匹配 |
| 情绪驱动型 | 量化模式 | 情绪50%+资金35%+消息面15% |
| 资金驱动型 | 量化模式 | 情绪50%+资金35%+消息面15% |
| 概念炒作型 | 量化模式 | 情绪50%+资金35%+消息面15% |
| 国际事件型 | 混合模式 | 智能匹配 |

### 步骤3：执行四维评分

调用02-scoring.md的评分公式：
```
总分 = 情绪到位×30% + 催化剂到位×25% + 逻辑强度×25% + 机构共识×20%
```

### 步骤4：构建作战地图

| 位置 | 定义 | 选择标准 |
|------|------|---------|
| **核心突击位** | 板块龙头，领涨先锋 | 涨幅前3+成交额前5+人气最高 |
| **侧翼观测位** | 跟随上涨，验证强度 | 涨幅3-8%+成交额中等+走势稳健 |
| **关键路标** | 需要关注的关键信号点 | 突破/确认/预警信号 |

### 步骤5：风险评估

引用07-risk.md的风险分级标准：
- 🟢低风险：风险分级1级
- 🟡中风险：风险分级2级
- 🟠高风险：风险分级3级
- 🔴极高风险：风险分级4级

## 三件套执行要求

### 苏格拉底式提问

每个热点必须质疑：
1. 我的假设是什么？
2. 有反例吗？
3. 逻辑有没有漏洞？
4. 换个角度会怎样？
5. 大爷会怎么质疑我？

### 辩论模式

每个热点必须做多空对撞：
- 多方逻辑：支撑上涨的理由
- 空方逻辑：压制上涨的理由
- 最终结论：多空博弈后的判断

### 预演失败

每个热点必须推演失败场景：
1. 失败场景描述
2. 触发阈值
3. 应对方案
