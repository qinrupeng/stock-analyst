## ⚠️ 核心规则（v5.0必读）

### 数据源优先级（已实测验证）

| 优先级 | 数据源 | 用途 | 准确度 |
|--------|--------|------|--------|
| ★★★ | **腾讯财经 qt.gtimg.cn** | 实时价格/涨幅/PE/换手/市值 | 100%✅ |
| ★★☆ | **Browser东方财富** | 资金流/研报/消息/板块 | 准 |
| ★☆☆ | 东方财富API | 技术指标/板块题材 | 公式可用 |

### 已废弃数据源（不再使用）
- ❌ 新浪财经hq接口 — 字段解析有歧义，容易把昨收/今开搞反
- ❌ 东方财富API f169/f170/f174 — 基准错误或数据失真
- ❌ 雪球API — 需要签名，无法自动化

### 第1步：腾讯财经拿实时数据
```bash
# 腾讯财经（★首选，Referer随意但要有）
Invoke-RestMethod -Uri "https://qt.gtimg.cn/q=sz301308" -Headers @{"Referer"="https://gu.qq.com"} | ConvertTo-Json -Depth 3
```

### 计算公式
```python
# 成交额（亿元）
成交额_亿 = 字段36 / 10000 / 100000000

# 总市值（亿元）
总市值_亿 = 字段47 / 10000 / 100000000

# 流通市值（亿元）
流通市值_亿 = 字段48 / 10000 / 100000000

# 涨跌幅验证
验证涨幅 = 字段3 - 字段4          # 应等于字段31
验证涨跌幅 = (字段3 - 字段4) / 字段4 * 100  # 应等于字段32
```

### 验证规则（每条数据必须验证）
```
□ 现价 - 昨收 = 涨跌额（字段3 - 字段4 = 字段31）
□ (现价 - 昨收) / 昨收 * 100 = 涨跌幅（字段32）
□ 现价 ≤ 最高， 现价 ≥ 最低
□ 涨停价 = 昨收 * 1.2（创业板），沪市主板 = *1.1
□ 字段3和字段9应该完全一致
```
