# 评分计算器模板

> 用于自动计算热点态势评级
> 版本：1.0 (2026-03-14)

---

## 评分公式

```
总分 = 消息面驱动(45分) + 市场预期(25分) + 情绪驱动(30分) = 100分
```

> **风险分级**：风险分级（🟢低/🟡中/🟠高/🔴极高）独立评估，不计入热点评分

## 使用说明

1. 打印或复制此模板
2. 根据热点逐项检查打分
3. 填写依据（必填，否则不计分）
4. 计算总分
5. 对照评级标准确定评级
